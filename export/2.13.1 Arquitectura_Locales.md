# 2.13.1 Arquitectura Locales

> No cargar textos en francés si el usuario está en México.

---

## Estructura de Archivos

Evitamos un archivo JSON gigante. Dividimos por dominio:

```
locales/
├── es-MX/
│   ├── auth.json
│   ├── dashboard.json
│   └── common.json
├── en-US/
│   ├── auth.json
│   └── ...
```

## Lazy Loading Configuration

En `nuxt.config.ts`, configuramos la carga bajo demanda:

```ts
i18n: {
  lazy: true,
  langDir: 'locales',
  defaultLocale: 'es-MX',
  locales: [
    { code: 'es-MX', file: 'es-MX/common.json' },
    { code: 'en-US', file: 'en-US/common.json' }
  ]
}
```

Esto asegura que el bundle inicial sea ligero y solo descargue el idioma necesario.

---

## Keys Anidadas

Usamos dot-notation para acceso type-safe:

```vue
<template>
  <h1>{{ $t('dashboard.welcome_message') }}</h1>
</template>
```
