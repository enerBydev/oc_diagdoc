# 6.1.4 Tipografía Google Sans Flex

> Sistema tipográfico variable para OnlyCar basado en **Google Sans Flex**.
> **Estilo "Tech Premium"** moderno y accesible.

---

## Visión General

**Google Sans Flex** es la fuente tipográfica principal de OnlyCar, con **6 ejes variables** que permiten máxima expresividad y control en cualquier contexto.

| Característica | Valor |
|----------------|-------|
| **Principal** | Google Sans Flex |
| **Secundaria** | Google Sans |
| **Formato** | Variable Font (WOFF2) |
| **Ejes** | 6 |
| **Presupuesto** | < 40 KB (WOFF2 subsetted) |
| **Licencia** | Apache 2.0 (Open Source) |

---

## Ejes Variables

| Eje | Tag | Rango | Descripción |
|-----|-----|-------|-------------|
| **Peso** | `wght` | 100-900 | De Thin a Black |
| **Ancho** | `wdth` | 75%-125% | De Condensed a Expanded |
| **Tamaño Óptico** | `opsz` | 8-144 | Optimización automática por tamaño |
| **Inclinación** | `slnt` | 0-12 | De recto a oblicuo |
| **Grade** | `GRAD` | -50 a 150 | Ajuste fino sin cambiar peso |
| **Redondez** | `ROND` | 0-100 | De Sharp a Friendly |

## Uso en Web (CSS)

### Cargar la Fuente (CDN Google)

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Google+Sans+Flex:wght@100..900&display=swap" rel="stylesheet">
```

### Variables CSS (Design Tokens)

```css
:root {
  --oc-font-sans: 'Google Sans Flex', 'Google Sans', system-ui, sans-serif;
  --oc-font-display: 'Google Sans Flex';
  --oc-font-text: 'Google Sans', sans-serif;
}
```

### Ejemplos de Uso

```css
/* Titular Principal */
.headline {
    font-family: var(--oc-font-display);
    font-variation-settings: 'wght' 800, 'opsz' 48, 'ROND' 20;
    color: #E87C1F; /* OnlyCar Orange */
}

/* Cuerpo de Texto */
.body-text {
    font-family: var(--oc-font-sans);
    font-variation-settings: 'wght' 400, 'opsz' 14, 'ROND' 50;
    color: #FFFFFF;
}

/* UI de Dashboard */
.dashboard-data {
    font-family: var(--oc-font-sans);
    font-variation-settings: 'wght' 500, 'wdth' 90, 'GRAD' 25;
}
```

## Uso en Audiovisual

Para After Effects, Premiere Pro u otros:

1. Descargar Google Sans Flex desde Google Fonts
2. Instalar el archivo `.ttf` variable
3. Usar sliders de Variable Font para animar ejes

Los ejes pueden animarse para crear transiciones dinámicas (ej: peso aumenta al ritmo del audio).

## Optimización de Performance

> [!IMPORTANT]
> Presupuesto de fuentes: **< 40 KB** (WOFF2 only)

### Estrategias de Optimización

| Estrategia | Ahorro | Implementación |
|------------|--------|----------------|
| **Subsetting** | ~50% | Solo glifos latinos + español |
| **CDN Google** | 100% cache | Cacheo global optimizado |
| **font-display: swap** | 0 FOIT | Evita flash de texto invisible |

### Carga CDN vs Self-Hosted

| Método | Tamaño | Ventaja |
|--------|--------|---------|
| **CDN Google** | ~0 KB (cached) | Recomendado para web |
| **Self-hosted** | ~35 KB (subsetted) | Control total |
