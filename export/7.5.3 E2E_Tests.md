# 7.5.3 E2E Tests

> Tests que simulan usuarios reales interactuando con la aplicación.

---

## Herramienta: Playwright

```typescript
// playwright.config.ts
export default defineConfig({
  testDir: './tests/e2e',
  use: {
    baseURL: 'http://localhost:3000',
    trace: 'on-first-retry'
  },
  projects: [
    { name: 'chromium' },
    { name: 'Mobile Chrome', use: devices['Pixel 5'] }
  ]
});
```

---

## Flujos Críticos

| Flujo | Pasos | Prioridad |
|-------|-------|-----------|
| Agendar Cita | Login → Servicio → Ubicación → Pago | P0 |
| Cancelar Cita | Ver citas → Cancelar → Confirmar | P1 |
| Login OAuth | Click Google → Authorize → Dashboard | P0 |
| Chat Operador | Abrir chat → Enviar mensaje → Recibir | P1 |

---

## Ejemplo

```typescript
test('flujo agendar cita completo', async ({ page }) => {
  await page.goto('/');
  await page.click('[data-testid="login-google"]');
  // ... OAuth flow
  
  await page.click('[data-testid="agendar-btn"]');
  await page.click('[data-testid="servicio-lavado-express"]');
  await page.fill('[data-testid="direccion"]', 'Calle Reforma 123');
  await page.click('[data-testid="confirmar-btn"]');
  
  await expect(page.locator('[data-testid="cita-confirmada"]')).toBeVisible();
});
```
