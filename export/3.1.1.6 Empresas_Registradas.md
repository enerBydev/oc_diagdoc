## Tipos de Contrato

| Tipo | Duración | Características |
|------|----------|-----------------|
| **Mensual** | 1 mes renovable | Flexibilidad, sin compromiso largo |
| **Anual** | 12 meses | Descuento adicional LEALTAD_ANUAL |
| **Proyecto** | Variable | Para eventos específicos |

---

## Formato de Identificadores

| Tipo | Formato | Ejemplo |
|------|---------|---------|
| Cliente B2C | `B2C-NLD-{4 dígitos}` | B2C-NLD-0001 |
| Empleado Corp | `B2C-CORP-{empresa}-{3 dígitos}` | B2C-CORP-BYD-001 |
| Empresa B2B | `B2B-{empresa}-NLD-{3 dígitos}` | B2B-BYD-NLD-001 |

---

## Modelo de Datos

```sql
CREATE TABLE empresas (
  id UUID PRIMARY KEY,
  codigo VARCHAR(20) UNIQUE NOT NULL,
  nombre VARCHAR(200) NOT NULL,
  activo BOOLEAN DEFAULT true,
  tipo_contrato VARCHAR(20), -- mensual, anual, proyecto
  corporate_plus_activo BOOLEAN DEFAULT false,
  prioridad_agenda_horas INTEGER DEFAULT 48,
  tarjetas_referido INTEGER DEFAULT 10
);
```
