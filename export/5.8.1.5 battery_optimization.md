## Optimizaciones Implementadas

### 1. Solo en estado `en_camino`
```typescript
// Tracking solo cuando necesario
if (estado === 'en_camino') startTracking();
if (estado === 'llegado') stopTracking();
```

### 2. Pausar al minimizar
```typescript
App.addListener('appStateChange', ({ isActive }) => {
  if (!isActive) stopTracking();
});
```

### 3. Throttle de broadcasts
