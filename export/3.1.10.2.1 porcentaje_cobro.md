# 3.1.10.2.1 Porcentaje de Cobro

Rango de deducción semanal: 10-25%.

---

## Asignación de Porcentaje

```typescript
const asignarPorcentajeCobro = (
  score: number,
  nivel: string
): number => {
  // Base por nivel
  const baseNivel = { bronce: 15, plata: 20, oro: 25, diamante: 25 };
  let porcentaje = baseNivel[nivel] || 15;
  
  // Ajuste por score
  if (score >= 90) porcentaje = Math.max(porcentaje, 25);
  else if (score < 70) porcentaje = Math.min(porcentaje, 15);
  
  // Rango permitido
  return Math.min(Math.max(porcentaje, 10), 25);
};
```

---

## Ejemplo de Deducción

```
Ganancias semana: $2,500
Porcentaje asignado: 15%
Deducción: $375
Pago neto: $2,125
```
