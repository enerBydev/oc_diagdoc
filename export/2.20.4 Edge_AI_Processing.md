# 2.20.4 Edge AI Processing

> **[FUTURO]** Inteligencia distribuida cerca del usuario, no en el servidor central.

---

## Concepto

Utilizar la red de **Cloudflare Workers AI** para ejecutar modelos pequeños (o pre-procesamiento) a milisegundos del usuario, en lugar de enviar todo a APIs costosas centrales.

---

## Funciones en el Edge

### 1. Compresión Inteligente de Imágenes
Antes de guardar en R2:
*   Convertir a WebP/AVIF.
*   Redimensionar a max 1920px.
*   Eliminar metadata irrelevante (pero preservar GPS/Date para auditoría en DB separada).

### 2. Sanitización de Inputs
*   Limpieza de texto antes de entrar a base de datos.
*   Detección básica de patrones de inyección SQL/XSS usando modelos ligeros de detección de anomalías.

### 3. Bot Detection Avanzado
*   Análisis de comportamiento de request (Fingerprinting).
*   Si parece bot, presentar Challenge (Turnstile) antes de permitir acceso a APIs de precios.

---

## Stack Tecnológico

*   **Cloudflare Workers AI:** Ejecución de modelos como Llama-2 o Whisper (Audio to Text) directo en el edge.
*   **Ventaja:** Costo mucho menor que OpenAI para tareas simples.
*   **Ejemplo:** Transcripción de notas de voz de chat (Audio -> Texto) para indexación y búsqueda.
