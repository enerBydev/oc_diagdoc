# 6.3.3 Cards

> Contenedores visuales para información agrupada.

---

## Variantes

| Variante | Uso | Contenido |
|----------|-----|-----------|
| **service** | Catálogo servicios | Imagen, nombre, precio, CTA |
| **package** | Catálogo paquetes | Lista servicios, precio total |
| **chat** | Mensaje chat | Avatar, nombre, preview, hora |
| **testimonial** | Reseñas | Avatar, nombre, rating, texto |
| **cita** | Agenda operador | Cliente, servicio, hora, ubicación |

---

## Anatomía Card Service

```
┌─────────────────────────┐
│      [Imagen 16:9]      │
├─────────────────────────┤
│ ■ Nombre Servicio       │
│ Descripción breve...    │
├─────────────────────────┤
│ $350 MXN    [Contratar] │
└─────────────────────────┘
```

## Implementación Nuxt UI

> [!IMPORTANT]
> Usar `<UCard>`. [Docs](https://ui.nuxt.com/components/card)

### Card Service

```vue
<UCard>
  <template #header>
    <img src="/servicios/lavado.jpg" class="w-full aspect-video object-cover" />
  </template>
  
  <h3 class="font-bold">Lavado Exterior</h3>
  <p class="text-neutral-400">Limpieza completa del exterior</p>
  
  <template #footer>
    <div class="flex justify-between items-center">
      <span class="text-xl font-bold text-primary">$350 MXN</span>
      <UButton label="Contratar" />
    </div>
  </template>
</UCard>
```

### Card Chat

```vue
<UCard class="cursor-pointer hover:bg-neutral-800">
  <div class="flex gap-3">
    <UAvatar src="/avatars/juan.jpg" />

      <div class="flex justify-between">
        <span class="font-medium">Juan Pérez</span>

      </div>
      <p class="text-sm text-neutral-400 truncate">Listo, ya terminé el servicio...</p>
    </div>
  </div>
</UCard>
```
