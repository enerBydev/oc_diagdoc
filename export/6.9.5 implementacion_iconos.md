## Implementación @nuxt/icon

> [!IMPORTANT]
> Usar `<Icon>` de @nuxt/icon. [Docs](https://nuxt.com/modules/icon)
> Acceso a 200,000+ iconos de Iconify.

### Uso Básico

```vue
<!-- Heroicons (default) -->
<Icon name="i-heroicons-home" />

<!-- Con tamaño y color -->
<Icon name="i-heroicons-check-circle" size="24" class="text-green-500" />

<!-- Otras colecciones -->
<Icon name="i-mdi-car" />           <!-- Material Design -->
<Icon name="i-lucide-calendar" />   <!-- Lucide -->
<Icon name="i-ph-chat-circle" />    <!-- Phosphor -->
```

### Config nuxt.config.ts

```ts
export default defineNuxtConfig({
  modules: ['@nuxt/icon'],
  icon: {
    size: '24px',
    class: 'icon',
    aliases: {
      'vehicle': 'i-heroicons-truck',
      'service': 'i-heroicons-sparkles',
      'payment': 'i-heroicons-credit-card'
    }
  }
})
```

### Iconos de Dominio OnlyCar

```vue
<Icon name="vehicle" />   <!-- Alias → i-heroicons-truck -->
<Icon name="service" />   <!-- Alias → i-heroicons-sparkles -->

```
