# 6.3.4 Modales

> Ventanas emergentes para acciones importantes.

---

## Variantes

| Variante | Uso | Acciones |
|----------|-----|----------|
| **confirmation** | Confirmar acción destructiva | Confirmar / Cancelar |
| **form** | Formularios rápidos | Submit / Cancelar |
| **lightbox** | Ver imágenes ampliadas | Cerrar / Navegar |
| **info** | Información importante | Solo cerrar |

---

## Anatomía

```
┌─────────────────────────────────┐
│ ✕                               │ ← Header + Close
├─────────────────────────────────┤
│                                 │
│     [Contenido del modal]       │ ← Body
│                                 │
├─────────────────────────────────┤
│        [Cancelar] [Confirmar]   │ ← Footer
└─────────────────────────────────┘
```

## Implementación Nuxt UI

> [!IMPORTANT]
> Usar `<UModal>`, `useModal()`. [Docs](https://ui.nuxt.com/components/modal)

### Modal Confirmación

```vue
<script setup>
const modal = useModal()
</script>

<template>
  <UButton @click="modal.open(ConfirmDeleteModal)" label="Eliminar" color="error" />
</template>

<!-- ConfirmDeleteModal.vue -->
<template>
  <UModal>
    <UCard>
      <template #header>
        <h3>¿Eliminar vehículo?</h3>
      </template>
      
      <p>Esta acción no se puede deshacer.</p>
      
      <template #footer>
        <div class="flex gap-2 justify-end">
          <UButton variant="ghost" label="Cancelar" @click="modal.close()" />
          <UButton color="error" label="Eliminar" @click="handleDelete" />
        </div>
      </template>
    </UCard>
  </UModal>
</template>
```
