# 1.3.6.1.4.2 Soundscape de Alertas

Sistema de sonidos personalizados para solicitudes.

---

## Filosofía

> [!IMPORTANT]
> **El sonido debe ser:**
> - **Distintivo:** NO confundirse con WhatsApp
> - **Urgente:** Transmitir oportunidad limitada
> - **Largo:** 8-15 segundos

---

## Tipos de Sonido

| Tipo | Duración | Descripción |
|------|----------|-------------|
| `service_request.ogg` | 10s | Estándar |
| `service_urgent.ogg` | 12s | Urgente |
| `service_favorite.ogg` | 8s | Cliente favorito |

---

## Control de Volumen

```kotlin
fun playServiceRequestSound(soundType: SoundType) {
    // Guardar estado actual
    previousVolume = audioManager.getStreamVolume(AudioManager.STREAM_NOTIFICATION)
    
    // Forzar volumen al 100%
    val maxVolume = audioManager.getStreamMaxVolume(AudioManager.STREAM_NOTIFICATION)
    audioManager.setStreamVolume(AudioManager.STREAM_NOTIFICATION, maxVolume, 0)
    
    // Cambiar a normal si está en silencio
    if (audioManager.ringerMode == AudioManager.RINGER_MODE_SILENT) {
        audioManager.ringerMode = AudioManager.RINGER_MODE_NORMAL
    }
    
    // Restaurar después de 15 segundos
    Handler(Looper.getMainLooper()).postDelayed({ restoreAudioSettings() }, 15000)
}
```
