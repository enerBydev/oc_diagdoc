# 3.1.10.1.4 Plazos de Pago

Duración del financiamiento.

---

## Plazos Disponibles

| Plazo | Cuota Semanal | Mejor para |
|-------|---------------|------------|
| 8 semanas | 12.5% monto | Artículos pequeños |
| 12 semanas | 8.3% monto | Equipo medio |
| 16 semanas | 6.25% monto | Equipo grande |
| 24 semanas | 4.2% monto | Kits completos |

---

## Cálculo de Cuota

```typescript
const calcularCuotaSemanal = (
  totalConInteres: number,
  plazoSemanas: number
): number => {
  return Math.ceil(totalConInteres / plazoSemanas);
};

// Ejemplo: Hidrolavadora $8,627.50 en 12 semanas
// Cuota semanal: $719/semana
```
