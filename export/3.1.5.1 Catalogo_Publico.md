# 3.1.5.1 Catálogo Público API

> Endpoints de lectura pública sin necesidad de autenticación.

---

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/catalogo/servicios` | Lista servicios activos con precios |
| GET | `/api/catalogo/paquetes` | Lista paquetes B2C activos |
| GET | `/api/catalogo/servicio/:codigo` | Detalle de un servicio |
| GET | `/api/catalogo/paquete/:codigo` | Detalle de un paquete |

---

## Ejemplo: GET /api/catalogo/servicios

```typescript
export default defineEventHandler(async (event) => {
  const client = await serverSupabaseClient(event);
  
  const { data, error } = await client
    .from('v_servicios_precios')
    .select('codigo, nombre, total_bloqueado, precio_b2c, descripcion')
    .order('orden');
  
  return { success: true, data, meta: { total: data.length, currency: 'MXN' } };
});
```
