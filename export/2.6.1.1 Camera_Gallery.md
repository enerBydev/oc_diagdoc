# 2.6.1.1 Camera & Gallery

> "El ojo del operador". Captura de fotos de alta calidad para documentar servicios.

---

## Flujo de Permisos

Antes de `Camera.getPhoto()`, verificamos permisos.

```ts
// composables/useCamera.ts
async function checkPermissions() {
  const status = await Camera.checkPermissions();
  if (status.camera !== 'granted') {
    const request = await Camera.requestPermissions();
    if (request.camera !== 'granted') throw new Error('Permisos denegados');
  }
}
```

### Android 13+
Se requiere permiso explícito para leer galería (`READ_MEDIA_IMAGES`) en lugar del antiguo `READ_EXTERNAL_STORAGE`.

---

## Configuración de Calidad

Para evitar saturar el ancho de banda, comprimimos en origen:

```ts
const image = await Camera.getPhoto({
  quality: 80, // Compresión 80% JPEG
  allowEditing: false, // No permitir recortar nativo (lento)
  resultType: CameraResultType.Uri, // Evitar Base64 en memoria
  saveToGallery: false // No ensuciar el carrete del usuario
});
```
