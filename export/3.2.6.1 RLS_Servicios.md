# 3.2.6.1 RLS Servicios

> Policies para servicios, paquetes, paquete_servicios.

---

## Políticas

```sql
-- Lectura pública de servicios activos
CREATE POLICY "servicios_read_public"
  ON servicios FOR SELECT
  USING (activo = true);

-- Solo admin puede modificar
CREATE POLICY "servicios_admin_all"
  ON servicios FOR ALL
  USING (auth.user_role() = 'admin');
```
