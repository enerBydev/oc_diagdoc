# 3.1.10.1.1 Elegibilidad

Requisitos para acceder a financiamiento.

---

## Requisitos Mínimos

| Requisito | Valor |
|-----------|-------|
| Servicios completados | ≥100 |
| Nivel mínimo | Bronce |
| Rating promedio | ≥4.5 |
| Sin deudas activas | ✅ |
| Caja al día | ✅ |
| Antigüedad | ≥3 meses |

---

## Cálculo de Score

```typescript
const calcularScoreFinanciamiento = (operador: Operador): number => {
  let score = 0;
  
  // Servicios (max 30 pts)
  if (operador.servicios >= 500) score += 30;
  else if (operador.servicios >= 300) score += 25;
  else if (operador.servicios >= 200) score += 20;
  else if (operador.servicios >= 100) score += 15;
  
  // Rating (max 25 pts)
  score += Math.min(operador.rating * 5, 25);
  
  // Nivel (max 20 pts)
  const puntosNivel = { nuevo: 0, bronce: 10, plata: 15, oro: 18, diamante: 20 };
  score += puntosNivel[operador.nivel];
  
  // Historial de pagos (max 15 pts)
  if (operador.pagosSinRetraso / operador.totalPagos >= 0.95) score += 15;
  else if (operador.pagosSinRetraso / operador.totalPagos >= 0.85) score += 10;
  
  // Antigüedad (max 10 pts)
  const meses = differenceInMonths(new Date(), operador.fechaActivacion);
  score += Math.min(meses, 10);
  
  return score; // Max 100
};

// Score ≥ 70: Elegible
// Score 50-69: Requiere revisión
// Score < 50: No elegible
```
