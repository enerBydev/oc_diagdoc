# 6.8.1 Transiciones

| Tipo | Duración | Easing |
|------|----------|--------|
| Page | 300ms | ease-out |
| Modal | 200ms | ease-in-out |
| Dropdown | 150ms | ease-out |

## Implementación @vueuse/motion

> [!IMPORTANT]
> Usar `v-motion` directives. [Docs](https://motion.vueuse.org/)

### Transición de Página

```vue
<template>
  <div
    v-motion
    :initial="{ opacity: 0, y: 20 }"
    :enter="{ opacity: 1, y: 0, transition: { duration: 300 } }"
    :leave="{ opacity: 0, y: -20 }"
  >
    <slot />
  </div>
</template>
```

### Transición de Modal

```vue
<div
  v-motion
  :initial="{ opacity: 0, scale: 0.95 }"
  :enter="{ opacity: 1, scale: 1, transition: { duration: 200 } }"
/>
```
