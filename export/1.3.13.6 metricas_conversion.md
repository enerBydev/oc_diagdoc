## Queries de Analytics

### Resumen Mensual

```sql
SELECT 
  DATE_TRUNC('month', enviado_at) AS mes,
  COUNT(*) AS enviados,
  COUNT(*) FILTER (WHERE abierto) AS abiertos,
  COUNT(*) FILTER (WHERE click_cta) AS clicks,
  COUNT(*) FILTER (WHERE convirtio) AS conversiones,
  ROUND(100.0 * COUNT(*) FILTER (WHERE convirtio) / COUNT(*), 2) AS tasa_conversion
FROM reminder_history
GROUP BY 1 ORDER BY 1 DESC;
```

### Por Template

```sql
SELECT 
  template_id,
  COUNT(*) AS enviados,
  ROUND(100.0 * COUNT(*) FILTER (WHERE convirtio) / COUNT(*), 2) AS tasa
FROM reminder_history
GROUP BY 1 ORDER BY tasa DESC;
```
