# 5.11.1 Point Terminal

Uso de terminales MercadoPago Point por operadores.

---

## Contexto

El operador puede adquirir su propia terminal MercadoPago Point para cobrar tarjetas en sitio. Este dinero va a **su cuenta personal de MercadoPago**, no a OnlyCar.

---

## Terminales Compatibles

| Modelo | Precio | Características |
|--------|--------|-----------------|
| **Point Plus** | ~$849 | Bluetooth, 8h batería |
| **Point Smart** | ~$1,899 | Pantalla táctil, WiFi |
| **Point Mini** | ~$399 | Básico, Bluetooth |

---

## Flujo de Cobro

```
Cliente quiere pagar con tarjeta física
        │
        ▼
Operador indica monto en terminal Point
        │
        ▼
Cliente pasa/inserta tarjeta
        │
        ▼
MercadoPago procesa transacción
        │
        ▼
Dinero entra a cuenta MP del OPERADOR
        │
        ▼
Sistema OnlyCar registra cobro en caja
        │
        ▼
Operador debe liquidar antes de 21:30
```

---

## Detección Automática (FUTURO)

> [!NOTE]
> **Por ahora:** Operador registra manualmente el cobro con tarjeta Point.
> **Futuro:** Integrar API MercadoPago para detectar cobros automáticamente.

### API Point (Futuro)

```typescript
// FUTURO: Webhook de cobro Point
const detectarCobroPoint = async (paymentId: string) => {
  const payment = await mercadopago.payment.get(paymentId);
  
  if (payment.payment_type === 'credit_card' && 
      payment.point_of_interaction?.type === 'MOBILE_READER') {
    // Es un cobro con Point del operador
    const operadorMPId = payment.collector_id;
    
    // Registrar en caja
    await registrarCobroTarjeta(operadorMPId, payment.transaction_amount);
  }
};
```

## Comisiones MercadoPago

| Tipo | Comisión |
|------|----------|
| Débito | 1.99% + IVA |
| Crédito | 3.15% + IVA |
| Sin contacto | Mismo que tipo tarjeta |

> [!TIP]
> El operador absorbe esta comisión al usar su propio Point.
> OnlyCar recibe el pago vía Checkout Pro (Saldo MP del operador).
