# 4.6.4 Endpoints Admin

> Endpoints exclusivos para rol Admin.

---

## Endpoints CRUD

| Método | Ruta | Descripción |
|--------|------|-------------|
| Método | Ruta | Descripción |
|--------|------|-------------|
| **Configuración** | | |
| GET | `/api/admin/config` | Obtener configuración global de precios |
| PUT | `/api/admin/config` | Actualizar configuración (JsonB) |
| **Catálogo** | | |
| GET | `/api/admin/servicios` | Listar todos los servicios (incluye inactivos) |
| POST | `/api/admin/servicios` | Crear nuevo servicio |
| PUT | `/api/admin/servicios/:id` | Editar servicio existente |
| DELETE | `/api/admin/servicios/:id` | Soft-delete servicio |
| GET | `/api/admin/paquetes` | Listar paquetes |
| POST | `/api/admin/paquetes` | Crear paquete |
| PUT | `/api/admin/paquetes/:id` | Editar paquete |
| DELETE | `/api/admin/paquetes/:id` | Soft-delete paquete |
| **Empresas B2B** | | |
| GET | `/api/admin/empresas` | Listar empresas registradas |
| POST | `/api/admin/empresas` | Onboarding nueva empresa |
| PUT | `/api/admin/empresas/:id` | Actualizar datos/contrato empresa |
| **Usuarios** | | |
| GET | `/api/admin/usuarios` | Listar usuarios sistema |
| PATCH | `/api/admin/usuario/:id` | Modificar rol/estado usuario |
| DELETE | `/api/admin/usuario/:id` | Banear usuario |
