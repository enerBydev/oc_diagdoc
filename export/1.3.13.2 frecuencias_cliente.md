# 1.3.13.2 Frecuencias por Tipo de Cliente

> Definición de frecuencias base y factores de ajuste.

---

## Frecuencias Base por Segmento

| Segmento | Frecuencia | Justificación |
|----------|------------|---------------|
| **B2C Individual** | 14-21 días | Uso personal, acumulación normal |
| **B2B Flotilla** | 7-10 días | Imagen corporativa, uso intensivo |
| **Corporate+** | Contrato | Puede ser semanal fijo |
| **Cliente Premium** | 10-14 días | Mayor frecuencia, clientes recurrentes |
| **Cliente Nuevo** | 21 días | No presionar, permitir experiencia |

---

## Factores de Ajuste

| Factor | Modificador | Ejemplo |
|--------|-------------|---------|
| **Época de lluvias** | -3 días | 14 → 11 días |
| **Cliente VIP** (5+ servicios) | -2 días | 14 → 12 días |
| **Sin respuesta previa** | +7 días | 14 → 21 días |
| **Temporada alta** (dic-ene) | -2 días | 14 → 12 días |
| **2+ recordatorios ignorados** | PAUSA | No enviar más |

---

## Fórmula de Cálculo

```typescript
function calcularFrecuenciaAjustada(cliente: Cliente): number {
  // Frecuencia base según tipo
  let frecuencia = obtenerFrecuenciaBase(cliente.tipo)
  
  // Ajustes por factores
  if (esEpocaLluvias()) {
    frecuencia -= 3
  }
  
  if (cliente.serviciosHistoricos >= 5) {
    frecuencia -= 2  // Cliente VIP
  }
  
  if (cliente.recordatoriosIgnorados > 0) {
    frecuencia += (cliente.recordatoriosIgnorados * 7)
  }
  
  if (esTemporadaAlta()) {
    frecuencia -= 2
  }
  
  // Mínimo 7 días, máximo 30 días
  return Math.max(7, Math.min(frecuencia, 30))
}

function obtenerFrecuenciaBase(tipo: TipoCliente): number {
  const bases: Record<TipoCliente, number> = {
    'b2c': 14,
    'b2b': 10,
    'corporate': 7,  // Según contrato
    'premium': 12,
    'nuevo': 21
  }
  return bases[tipo] || 14
}
```

## Escenarios de Ejemplo

### Escenario 1: Cliente B2C Normal

| Factor | Valor |
|--------|-------|
| Base | 14 días |
| Época lluvias | No |
| Servicios previos | 2 |
| Ignorados | 0 |
| **Resultado** | **14 días** |

### Escenario 2: Cliente VIP en Temporada Alta

| Factor | Valor |
|--------|-------|
| Base | 14 días |
| Época lluvias | No |
| Servicios previos | 8 (VIP: -2) |
| Temporada alta | Sí (-2) |
| **Resultado** | **10 días** |

### Escenario 3: Cliente que Ignoró 1 Recordatorio

| Factor | Valor |
|--------|-------|
| Base | 14 días |
| Ignorados | 1 (+7) |
| **Resultado** | **21 días** |
