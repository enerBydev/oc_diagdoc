# 2.3.1 Google Maps API

> Configuración de Google Cloud Console para Maps Platform.

---

## Claves de API (Keys)

### 1. Browser Key (Frontend)
- **Uso**: JavaScript API loading (Mapas en web), Places Autocomplete.
- **Restricción**: Referer HTTP (`https://onlycar.mx/*`, `http://localhost:3000/*`).
- **Value**: `AIzaSyD...` (Ver Dashboard)

### 2. Android SDK Key
- **Uso**: Mapas nativos en la app Android.
- **Restricción**: Package Name (`mx.onlycar.app`) + SHA-1 Fingerprint.
- **Value**: `AIzaSyB...`

### 3. iOS SDK Key
- **Uso**: Mapas nativos en la app iOS.
- **Restricción**: Bundle ID (`mx.onlycar.app`).
- **Value**: `AIzaSyC...`

### 4. Server Key (Backend)
- **Uso**: Geocoding API, Directions API (cálculo rutas), Distance Matrix.
- **Restricción**: IP Address (Solo IPs de Cloudflare/Vercel).
- **Value**: `AIzaSyE...`

---

## APIs Habilitadas

1.  Maps JavaScript API
2.  Maps SDK for Android
3.  Maps SDK for iOS
4.  Geocoding API (Conversión LatLong <-> Dirección)
5.  Directions API (Cálculo de rutas y tiempo estimado)
6.  Places API (New)

---

## Cuotas y Límites

- **Presupuesto Mensual**: $200 USD (Crédito gratuito Google).
- **Alerta**: Al llegar al 80% del presupuesto ($160).
- **Hard Limit**: Detener servicio si excede $300 (evitar sorpresas).
