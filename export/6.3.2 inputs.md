# 6.3.2 Inputs

> Campos de entrada para formularios.

---

## Tipos

| Tipo | HTML | Uso |
|------|------|-----|
| **text** | `<input type="text">` | Texto general |
| **email** | `<input type="email">` | Correo electrónico |
| **password** | `<input type="password">` | Contraseñas |
| **tel** | `<input type="tel">` | Teléfono |
| **number** | `<input type="number">` | Valores numéricos |
| **date** | `<input type="date">` | Fechas |
| **select** | `<select>` | Opciones predefinidas |
| **file** | `<input type="file">` | Subir archivos |

## Implementación Nuxt UI

> [!IMPORTANT]
> Usar `<UInput>`, `<USelect>`, `<UFormGroup>`. [Docs](https://ui.nuxt.com/components/input)

### Input Básico

```vue
<UInput v-model="email" placeholder="correo@ejemplo.com" />
```

### Con Label y Error

```vue
<UFormGroup label="Email" :error="emailError">
  <UInput v-model="email" type="email" />
</UFormGroup>
```

### Password con Icono

```vue
<UInput 
  v-model="password" 
  type="password" 
  :ui="{ leading: { icon: { name: 'i-heroicons-lock-closed' } } }"
/>

<!-- O usando leading slot -->
<UInput v-model="password" type="password">
  <template #leading>
    <Icon name="i-heroicons-lock-closed" />
  </template>
</UInput>
```

### Number

```vue
<UInput 
  v-model="cantidad" 
  type="number" 
  :min="1" 
  :max="10"
  placeholder="Cantidad"
/>
```

### Select

```vue
<USelect 
  v-model="vehiculo" 
  :options="vehiculos"
  placeholder="Selecciona un vehículo"
/>

<script setup>
const vehiculos = [
  { label: 'Honda Civic', value: 'civic' },
  { label: 'Toyota RAV4', value: 'rav4' }
]
</script>
```

### Date

```vue
<!-- Input nativo -->
<UInput type="date" v-model="fecha" />

<!-- Para datepicker avanzado, usar librería externa -->
<!-- npm install @vuepic/vue-datepicker (agregar a 2.2. Docs.md) -->
```

### File Upload

```vue
<UInput type="file" accept="image/*" @change="handleFile" />

<script setup>
function handleFile(event: Event) {
  const target = event.target as HTMLInputElement
  const file = target.files?.[0]
  if (file) {
    console.log('Archivo seleccionado:', file.name)
  }
}
</script>
```

---

### Validación con Vuelidate

```vue
<UFormGroup 
  label="Teléfono" 
  :error="v$.phone.$errors[0]?.$message"
>
  <UInput v-model="phone" type="tel" />
</UFormGroup>
```
