# 4.1.4 Restricciones

> Limitaciones que condicionan el diseño del sistema.

---

## Restricciones Técnicas

| Restricción               | Razón             | Impacto                   |
| ------------------------- | ----------------- | ------------------------- |
| **Nuxt v4.x + Capacitor** | Stack definido    | Desarrollo mobile híbrido |
| **Supabase PostgreSQL**   | BaaS seleccionado | Sin cambio de DB          |
| **Cloudflare**            | Hosting edge      | Limitaciones Workers      |
| **PWA obligatorio**       | Offline-first     | Mayor complejidad         |

---

## Restricciones Legales (México)

| Normativa | Aplicación |
|-----------|------------|
| **LFPDPPP** | Protección datos personales |
| **CFF** | Código Fiscal (facturación) |
| **CFDI 4.0** | Factura electrónica |
| **Ley Fintech** | Si aplica procesamiento pagos |
| **GDPR** | Si hay usuarios EU (futuro) |

→ Ver cumplimiento: [[Proyecto OnlyCarNLD/Datos/5.4. cumplimiento_legal_fiscal]]
