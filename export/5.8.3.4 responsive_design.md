## Breakpoints

| Dispositivo | Ancho | Altura Mapa |
|-------------|-------|-------------|
| Mobile | < 640px | 200px |
| Tablet | 640-1024px | 250px |
| Desktop | > 1024px | 300px |

---

## Implementaci√≥n

```vue
<template>
  <div class="map-wrapper" :style="mapStyle">
    <MapaUbicacion v-bind="props" height="100%" />
  </div>
</template>

<script setup lang="ts">
const breakpoint = useBreakpoints();

const mapStyle = computed(() => ({
  height: breakpoint.value === 'mobile' ? '200px' 
        : breakpoint.value === 'tablet' ? '250px' 
        : '300px',
  borderRadius: '12px',
  overflow: 'hidden'
}));
</script>
```
