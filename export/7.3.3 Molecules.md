# 7.3.3 Molecules

> Combinaciones de atoms que forman componentes funcionales simples.

---

## Catálogo de Molecules

| Componente | Atoms Usados | Propósito |
|------------|--------------|-----------|
| `OcSearchBar` | OcInput + OcIcon + OcButton | Búsqueda con acción |
| `OcFormField` | OcInput + Label + Error | Campo de formulario |
| `OcNavItem` | OcIcon + Text + OcBadge | Item de navegación |
| `OcRating` | OcIcon (stars) + Text | Calificación |
| `OcPriceTag` | Text + OcBadge | Precio con descuento |

---

## Ejemplo: OcFormField

```vue
<!-- components/molecules/OcFormField.vue -->
<template>
  <div class="oc-form-field">
    <label 
      v-if="label"
      :for="id"
      class="oc-form-field__label"
    >
      {{ label }}
      <span v-if="required" class="oc-form-field__required">*</span>
    </label>
    
    <OcInput
      :id="id"
      v-model="model"
      :type="type"
      :placeholder="placeholder"
      :error="!!error"
      :disabled="disabled"
    />
    
    <p v-if="error" class="oc-form-field__error">
      {{ error }}
    </p>
    
    <p v-else-if="hint" class="oc-form-field__hint">
      {{ hint }}
    </p>
  </div>
</template>

<script setup lang="ts">
const model = defineModel<string>();

defineProps<{
  id: string;
  label?: string;
  type?: string;
  placeholder?: string;
  error?: string;
  hint?: string;
  required?: boolean;
  disabled?: boolean;
}>();
</script>
```
