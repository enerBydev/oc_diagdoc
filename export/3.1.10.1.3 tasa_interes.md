# 3.1.10.1.3 Tasa de Interés

Configuración de tasas por perfil.

---

## Rango de Tasas

| Score | Tasa Mensual | Tasa Anual Aprox |
|-------|--------------|------------------|
| 90-100 | 0.5% | 6% |
| 80-89 | 1.0% | 12% |
| 70-79 | 1.5% | 18% |
| 60-69 | 2.0% | 24% |
| 50-59 | 3.0% | 36% |

> [!NOTE]
> Tasas competitivas comparadas con financieras tradicionales (40-80% anual).

---

## Cálculo de Tasa

```typescript
const getTasaPorScore = (score: number): number => {
  if (score >= 90) return 0.5;
  if (score >= 80) return 1.0;
  if (score >= 70) return 1.5;
  if (score >= 60) return 2.0;
  return 3.0;
};

const calcularTotalConInteres = (
  monto: number,
  tasaMensual: number,
  plazoSemanas: number
): number => {
  const plazoMeses = plazoSemanas / 4;
  const interes = monto * (tasaMensual / 100) * plazoMeses;
  return monto + interes;
};
```
