## Implementación Nuxt UI

> [!IMPORTANT]
> Usar `useToast()`, `<UAlert>`, `<USkeleton>`. [Docs](https://ui.nuxt.com/components/toast)

### Toast

```vue
<script setup>
const toast = useToast()

function showSuccess() {
  toast.add({
    title: '¡Servicio completado!',
    color: 'success',
    icon: 'i-heroicons-check-circle'
  })
}

function showError() {
  toast.add({
    title: 'Error al guardar',
    description: 'Intenta de nuevo',
    color: 'error'
  })
}
</script>
```

### Alert

```vue
<UAlert color="warning" title="Tu sesión expirará en 5 minutos" />
```

### Skeleton

```vue
<template>
  <div v-if="loading" class="space-y-4">
    <USkeleton class="h-4 w-3/4" />
    <USkeleton class="h-4 w-1/2" />
  </div>
  <div v-else>{{ content }}</div>
</template>
```
