# 4.2.2 RF Cliente B2B

> Requisitos funcionales para clientes empresariales.

---

## Portal Empresa

| ID | Requisito | Prioridad |
|----|-----------|-----------|
| RF-B2B-001 | La empresa tiene dashboard dedicado | Alta |
| RF-B2B-002 | El admin empresa puede ver contratos | Alta |
| RF-B2B-003 | El admin empresa puede ver facturas | Alta |

## Gestión de Flotilla

| ID | Requisito | Prioridad |
|----|-----------|-----------|
| RF-B2B-010 | La empresa puede registrar vehículos de flotilla | Alta |
| RF-B2B-011 | Cada vehículo tiene placa + asignación | Alta |
| RF-B2B-012 | Agendamiento masivo de servicios | Media |

## Empleados Corporate+

| ID | Requisito | Prioridad |
|----|-----------|-----------|
| RF-B2B-020 | La empresa puede dar de alta empleados | Alta |
| RF-B2B-021 | El empleado recibe código de activación | Alta |
| RF-B2B-022 | La empresa puede ver uso por empleado | Media |

## Reportes

| ID | Requisito | Prioridad |
|----|-----------|-----------|
| RF-B2B-030 | Reporte de servicios por período | Alta |
| RF-B2B-031 | Reporte de gasto por vehículo | Media |
| RF-B2B-032 | Exportar a Excel/PDF | Media |
