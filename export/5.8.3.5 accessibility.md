# 5.8.3.5 Accessibility

Accesibilidad del mapa para todos los usuarios.

---

## Requerimientos WCAG

| Criterio | Implementación |
|----------|----------------|
| **1.1.1 Non-text** | Texto alternativo para markers |
| **2.1.1 Keyboard** | Navegación por teclado |
| **2.4.7 Focus Visible** | Indicador de foco |
| **4.1.2 Name, Role** | ARIA labels |

---

## ARIA Labels

```vue
<template>
  <div 
    ref="mapContainer"
    role="application"
    aria-label="Mapa mostrando ubicación del operador"
    tabindex="0"
    @keydown="handleKeyNav"
  >
    <span class="sr-only" aria-live="polite">
      {{ ariaDescription }}
    </span>
  </div>
</template>

<script setup lang="ts">
const ariaDescription = computed(() => {
  if (!operadorPosition.value) {
    return 'Esperando ubicación del operador';
  }
  return `Operador a ${distancia.value} metros de distancia, actualizado ${timeAgo.value}`;
});
</script>
```

## Navegación por Teclado

```typescript
const handleKeyNav = (e: KeyboardEvent) => {
  switch (e.key) {
    case 'ArrowUp':
      map.panBy([0, -50]);
      break;
    case 'ArrowDown':
      map.panBy([0, 50]);
      break;
    case 'ArrowLeft':
      map.panBy([-50, 0]);
      break;
    case 'ArrowRight':
      map.panBy([50, 0]);
      break;
    case '+':
    case '=':
      map.zoomIn();
      break;
    case '-':
      map.zoomOut();
      break;
    case 'Escape':
      map.getContainer().blur();
      break;
  }
};
```
