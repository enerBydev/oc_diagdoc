# 1.3.6.1.4.1 Notificaciones Premium

Sistema de notificaciones de alto impacto para solicitudes de servicio.

---

## Filosofía

> [!CAUTION]
> **Las solicitudes NO son notificaciones normales.**
> Son **oportunidades de trabajo** que el operador NO puede perderse.
> - Si está conduciendo, debe escucharla
> - Si tiene el teléfono en la bolsa, debe sentirla
> - **NUNCA** debe confundirse con WhatsApp

---

## Comparación

| Aspecto | Normal | **Premium** |
|---------|--------|-------------|
| Sonido | 1 segundo | 8-15 segundos |
| Vibración | 1 pulso | Patrón prolongado |
| Duración | Desaparece 3s | **Persiste** |
| Volumen | Respeta silencio | **Anula silencio** |
| Visual | Banner pequeño | **Pantalla completa** |

---

## Android: High-Priority Notification

```kotlin
val channel = NotificationChannel(CHANNEL_ID, CHANNEL_NAME, 
    NotificationManager.IMPORTANCE_HIGH).apply {
    setBypassDnd(true)  // Ignorar No Molestar
    enableVibration(true)
    vibrationPattern = longArrayOf(0, 500, 200, 500, 200, 500)
}

val notification = NotificationCompat.Builder(context, CHANNEL_ID)
    .setPriority(NotificationCompat.PRIORITY_MAX)
    .setCategory(NotificationCompat.CATEGORY_CALL)
    .setFullScreenIntent(fullScreenPendingIntent, true)
    .setAutoCancel(false)
    .setOngoing(true)
    .setTimeoutAfter(300000) // 5 minutos
    .build()
```
