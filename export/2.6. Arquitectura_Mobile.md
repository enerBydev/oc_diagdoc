# 2.6 Arquitectura Mobile (Capacitor)

> Puente nativo para compilar la PWA como apps de iOS y Android.

---

## Visión General

OnlyCar utiliza **Capacitor 6** para envolver la aplicación web Nuxt en un contenedor nativo (Webview) que permite acceso a hardware del dispositivo (GPS, Cámara, Push Notifications).

## Estrategia "Mobile First"

1. **Un solo código base:** `pages/` en Nuxt.
2. **Capacidades Progressivas:** Si `Capacitor.isNative` es true, usar plugins. Si no, fallback a Web API.
3. **Despliegue Atómico:** El backend (Supabase) es independiente de la versión de la app instalada.
