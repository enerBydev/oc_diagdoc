# 1.1.5 Reglas Comerciales

## Reglas de Contratos B2B

| Regla | Descripción |
|-------|-------------|
| **RC-001** | Todo cliente B2B debe firmar contrato antes del primer servicio |
| **RC-002** | Los contratos tienen fecha de vencimiento obligatoria |
| **RC-003** | Al renovar, se pueden modificar términos, frecuencia y precios |
| **RC-004** | La firma se realiza digitalmente vía Mifiel (e.firma/FIEL) |
| **RC-005** | Los contratos firmados se almacenan 5+ años (R2 encriptado) |

---

## Reglas de Precios

| Regla | Descripción |
|-------|-------------|
| **RP-001** | Los precios públicos incluyen IVA (absorbido) |
| **RP-002** | Factor B2C: 0.7 × precioBase |
| **RP-003** | Factor B2B: 0.45 × precioBase |
| **RP-004** | Redondeo al múltiplo de $10 más cercano |
| **RP-005** | Precios B2B NO se muestran públicamente |

---

## Reglas de Descuentos

| Regla      | Descripción                                               |
| ---------- | --------------------------------------------------------- |
| **RD-001** | BIENVENIDA30: Solo primera compra, incompatible con otros |
| **RD-002** | BIENVENIDA_REFERIDA: Solo si viene de referido            |
| **RD-003** | CORP15: Requiere verificación de email corporativo        |
| **RD-004** | Descuentos de volumen B2B son acumulables                 |
| **RD-005** | Fórmula acumulada: `1 - ((1-d1) × (1-d2) × ... × (1-dn))` |
| **RD-006** | **Descuentos SOLO aplican a PAQUETES**, NO a servicios individuales |

> [!IMPORTANT]
> Los servicios individuales NO tienen descuentos.
> Los descuentos (BIENVENIDA30, CORP15, etc.) son exclusivos para PAQUETES.

---

## Reglas de Servicios

| Regla | Descripción |
|-------|-------------|
| **RS-001** | Servicios individuales sin contrato disponibles para B2B |
| **RS-002** | Paquetes sin contrato tienen menor beneficio que con contrato |
| **RS-003** | Empleados B2B pueden solicitar servicios B2C personales |
| **RS-004** | La duración incluye buffer de agenda (15 min típico) |
| **RS-005** | Fotos antes/después son obligatorias para todos los servicios |

---

## Reglas de Facturación

| Regla | Descripción |
|-------|-------------|
| **RF-001** | Toda venta genera factura (cliente o "Público en General") |
| **RF-002** | Gigstack procesa 100% de la facturación CFDI 4.0 |
| **RF-003** | Autofactura disponible hasta 72 hrs post-servicio |
| **RF-004** | Conservación de CFDI: 5 años (Código Fiscal) |
| **RF-005** | B2B: Factura mensual consolidada automática |
