## Protocolo de Seguridad

1.  **Rotación**: Las claves deben rotarse cada 90 días o ante sospecha de compromiso.
2.  **Alcance**: Usar siempre claves con alcance restringido (Principio de menor privilegio).
3.  **Almacenamiento**: Nunca hardcodear en código fuente. Usar variables de entorno (Ver [[Proyecto OnlyCarNLD/Datos/2.4. variables_entorno]]).
