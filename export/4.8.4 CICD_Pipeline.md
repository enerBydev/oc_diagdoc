# 4.8.4 CI/CD Pipeline

> GitHub Actions + Cloudflare Pages.

---

## Flujo

```mermaid
graph LR
    A[Push] --> B[Lint]
    B --> C[Type Check]
    C --> D[Test]
    D --> E[Build]
    E --> F{Branch?}
    F -->|develop| G[Deploy Staging]
    F -->|main| H[Deploy Prod]
```

---

## Jobs

| Job | Trigger | Duración |
|-----|---------|----------|
| lint | PR, push | ~30s |
| typecheck | PR, push | ~45s |
| test | PR, push | ~2m |
| build | merge | ~3m |
| deploy | merge | ~1m |
