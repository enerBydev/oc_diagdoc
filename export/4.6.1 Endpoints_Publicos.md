# 4.6.1 Endpoints Públicos

> Endpoints accesibles sin token de autenticación.

---

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/catalogo/servicios` | Catálogo completo de servicios con precios |
| GET | `/api/catalogo/paquetes` | Paquetes activos (B2C) |
| GET | `/api/catalogo/servicio/:codigo` | Detalle profundo de un servicio |
| GET | `/api/catalogo/paquete/:codigo` | Detalle profundo de un paquete |
| POST | `/api/auth/login` | Login (Email/Pass o OAuth) |
| POST | `/api/auth/register` | Registro de nuevos usuarios |
| POST | `/api/webhooks/stripe` | Webhook de pagos Stripe |
| POST | `/api/webhooks/gigstack` | Webhook de facturación Gigstack |

---

## Rate Limiting

| Endpoint | Límite |
|----------|--------|
| `/api/public/*` | 100 req/min |
| `/api/auth/*` | 10 req/min |
| `/api/webhooks/*` | 1000 req/min |
