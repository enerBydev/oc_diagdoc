# 7.3.1 Design Tokens

> Variables fundamentales que definen la identidad visual del sistema.

---

## Colores

```css
:root {
  /* Brand */
  --oc-primary: #3B82F6;
  --oc-primary-hover: #2563EB;
  --oc-secondary: #10B981;
  --oc-secondary-hover: #059669;
  
  /* Semantic */
  --oc-success: #22C55E;
  --oc-warning: #F59E0B;
  --oc-error: #EF4444;
  --oc-info: #3B82F6;
  
  /* Neutral */
  --oc-gray-50: #F9FAFB;
  --oc-gray-100: #F3F4F6;
  --oc-gray-500: #6B7280;
  --oc-gray-900: #111827;
  
  /* Backgrounds */
  --oc-bg-primary: #FFFFFF;
  --oc-bg-secondary: #F3F4F6;
  --oc-bg-dark: #1F2937;
}
```

---

## Tipografía

```css
:root {
  /* Font Family - Google Sans Flex System */
  --oc-font-sans: 'Google Sans Flex', 'Google Sans', system-ui, sans-serif;
  --oc-font-display: 'Google Sans Flex';
  --oc-font-text: 'Google Sans', sans-serif;
  --oc-font-mono: 'JetBrains Mono', monospace;
  
  /* Font Sizes */
  --oc-text-xs: 0.75rem;   /* 12px */
  --oc-text-sm: 0.875rem;  /* 14px */
  --oc-text-base: 1rem;    /* 16px */
  --oc-text-lg: 1.125rem;  /* 18px */
  --oc-text-xl: 1.25rem;   /* 20px */
  --oc-text-2xl: 1.5rem;   /* 24px */
  --oc-text-3xl: 1.875rem; /* 30px */
  
  /* Font Weights */
  --oc-font-normal: 400;
  --oc-font-medium: 500;
  --oc-font-semibold: 600;
  --oc-font-bold: 700;
}
```

---

## Spacing

```css
:root {
  --oc-space-0: 0;
  --oc-space-1: 0.25rem;  /* 4px */
  --oc-space-2: 0.5rem;   /* 8px */
  --oc-space-3: 0.75rem;  /* 12px */
  --oc-space-4: 1rem;     /* 16px */
  --oc-space-6: 1.5rem;   /* 24px */
  --oc-space-8: 2rem;     /* 32px */
  --oc-space-12: 3rem;    /* 48px */
}
```
