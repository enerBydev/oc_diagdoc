# 3.1.7.1.4 Gastos Indirectos

Costos operativos que no son directamente del servicio.

---

## Categorías de Gastos Indirectos

| Categoría            | % del Ingreso | Descripción          |
| -------------------- | ------------- | -------------------- |
| **Pasarela de pago** | 3.6%          | Stripe, MercadoPago  |
| **Infraestructura**  | 2%            | Servers, apps, BD    |
| **Soporte**          | 1.5%          | Atención cliente     |
| **Marketing**        | 3%            | Adquisición clientes |
| **Administración**   | 2%            | Contabilidad, legal  |
| **Imprevistos**      | 1%            | Reserva contingencia |
| **TOTAL**            | **13.1%**     | —                    |

---

## Aplicación del Factor

```typescript
const FACTOR_GASTOS_INDIRECTOS = 0.131; // 13.1%

const calcularGastosIndirectos = (precioServicio: number): number => {
  return precioServicio * FACTOR_GASTOS_INDIRECTOS;
};

// Ejemplo: Lavado $290
// Gastos indirectos: $290 * 0.131 = $37.99
```

## Modelo de Datos

```sql
CREATE TABLE gastos_indirectos (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  categoria VARCHAR(50) NOT NULL UNIQUE,
  porcentaje DECIMAL(5,2) NOT NULL,
  descripcion TEXT,
  activo BOOLEAN DEFAULT true,
  updated_at TIMESTAMPTZ DEFAULT now()
);

-- Vista para total
CREATE VIEW v_factor_indirectos AS
SELECT SUM(porcentaje) / 100 as factor
FROM gastos_indirectos
WHERE activo = true;
```
