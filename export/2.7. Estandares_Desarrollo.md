# 2.7 Estándares de Desarrollo

> Gobernanza técnica para asegurar consistencia, calidad y mantenibilidad del código.

> [!NOTE]
> Para **diagramas de Git Flow** y Conventional Commits, ver [[Proyecto OnlyCarNLD/Datos/7.6 Git_Flow_DevOps]].

---

## Visión General

El desarrollo en OnlyCar sigue metodologías estrictas para garantizar:
1. **Consistencia:** Todo el código parece escrito por una sola persona.
2. **Trazabilidad:** Historial de Git limpio y semántico.
3. **Automatización:** CI/CD para detectar errores antes de producción.

## Tooling Obligatorio

Todos los desarrolladores deben tener configurado:
- **VS Code** con extensiones recomendadas (Vetur/Volar, ESLint).
- **Node.js** v20 LTS.
- **Git** configurado con Email corporativo.
