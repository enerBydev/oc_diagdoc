# 4.2.3 RF Corporate+

> Requisitos funcionales para empleados con beneficios Corporate+.

---

## Activación

| ID | Requisito | Prioridad |
|----|-----------|-----------|
| RF-CORP-001 | El empleado activa cuenta con código de empresa | Alta |
| RF-CORP-002 | El sistema valida dominio de email corporativo | Alta |
| RF-CORP-003 | El empleado hereda beneficios B2C + Corp+ | Alta |

## Beneficios

| ID | Requisito | Prioridad |
|----|-----------|-----------|
| RF-CORP-010 | Descuento permanente 15% (CORP15) | Alta |
| RF-CORP-011 | Prioridad en agenda (48h antes) | Media |
| RF-CORP-012 | 10 tarjetas físicas de referido | Media |
| RF-CORP-013 | Facturación a nombre de empresa | Alta |

## Uso de Beneficio

| ID | Requisito | Prioridad |
|----|-----------|-----------|
| RF-CORP-020 | El empleado puede usar beneficio para vehículo personal | Alta |
| RF-CORP-021 | El descuento se aplica automáticamente | Alta |
| RF-CORP-022 | La empresa puede ver uso del beneficio | Media |
