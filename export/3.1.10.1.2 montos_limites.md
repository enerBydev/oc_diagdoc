# 3.1.10.1.2 Montos y Límites

Límites de financiamiento por nivel.

---

## Límites por Nivel

| Nivel | Monto Máximo | Cuota Máx/Semana |
|-------|--------------|------------------|
| Bronce | $5,000 | 15% |
| Plata | $10,000 | 20% |
| Oro | $20,000 | 25% |
| Diamante | $35,000 | 25% |

---

## Cálculo de Cuota Máxima

```typescript
const calcularCuotaMaxima = (
  operador: Operador,
  ingresoPromedioSemanal: number
): number => {
  const porcentajeMax = PORCENTAJE_MAX_NIVEL[operador.nivel]; // 15-25%
  return ingresoPromedioSemanal * (porcentajeMax / 100);
};

// Ejemplo: Operador Plata, $2,500/semana promedio
// Cuota máxima: $2,500 * 0.20 = $500/semana
```
