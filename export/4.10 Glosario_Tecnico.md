# 4.10 Glosario Técnico

> Definiciones de términos clave del sistema OnlyCar.

---

## Términos de Negocio

| Término | Definición |
|---------|------------|
| **B2C** | Business-to-Consumer. Cliente individual que paga por sus propios servicios. |
| **B2B** | Business-to-Business. Empresa con contrato para flotilla de vehículos. |
| **Corporate+** | Empleado de empresa B2B que recibe beneficio como prestación laboral. |
| **Operador** | Persona que ejecuta los servicios de lavado/detailing. |
| **Cita** | Agendamiento de un servicio en fecha, hora y ubicación específica. |
| **Servicio** | Tipo de trabajo a realizar (Lavado Express, Detailing, etc.). |
| **Paquete** | Combinación de servicios con precio especial. |
| **Flotilla** | Conjunto de vehículos de una empresa B2B. |

---

## Términos Técnicos

| Término | Definición |
|---------|------------|
| **SSR** | Server-Side Rendering. Renderizado en servidor para SEO. |
| **PWA** | Progressive Web App. App web instalable. |
| **Edge** | Servidores distribuidos globalmente cerca de usuarios. |
| **Workers** | Funciones serverless en edge de Cloudflare. |
| **RLS** | Row Level Security. Seguridad a nivel de fila en Supabase. |
| **CFDI** | Comprobante Fiscal Digital por Internet. Factura electrónica SAT. |
| **PAC** | Proveedor Autorizado de Certificación (timbrado facturas). |
| **ADR** | Architecture Decision Record. Documento de decisión arquitectónica. |

---

| SLI | Service Level Indicator |

| JWT | JSON Web Token |

---

## Estados de Cita

| Estado | Significado |
|--------|-------------|
| `pendiente` | Creada, esperando operador |
| `aceptada` | Operador confirmó |
| `en_camino` | Operador en tránsito |
| `en_progreso` | Servicio en ejecución |
| `completada` | Servicio finalizado |
| `cancelada` | Cancelada por cliente u operador |

---

## Roles de Usuario

| Rol | Permisos |
|-----|----------|
| `cliente_b2c` | Agendar, pagar, calificar |
| `cliente_b2b` | Gestionar flotilla, ver reportes |
| `corporate_plus` | Agendar con beneficio empresa |
| `operador` | Ejecutar servicios, cobrar |
| `admin` | Acceso total al sistema |
