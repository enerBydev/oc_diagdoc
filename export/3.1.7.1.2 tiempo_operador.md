# 3.1.7.1.2 Tiempo del Operador

Costo del tiempo de trabajo del operador.

---

## Cálculo del Valor por Minuto

```typescript
// Ingreso objetivo mensual operador: $15,000
// Horas de trabajo estimadas: 160/mes
// Minutos totales: 9,600

const INGRESO_MENSUAL_OBJETIVO = 15000;
const HORAS_MENSUALES = 160;
const MINUTOS_MENSUALES = HORAS_MENSUALES * 60;

const VALOR_MINUTO_OPERADOR = INGRESO_MENSUAL_OBJETIVO / MINUTOS_MENSUALES;
// = $1.5625 por minuto
```

## Factores de Ajuste

| Factor | Multiplicador | Aplica cuando |
|--------|---------------|---------------|
| Vehículo grande | 1.2x | SUV, camioneta |
| Estado muy sucio | 1.3x | Lodo, tierra excesiva |
| Ubicación lejana | +$X | Traslado extra |
| Horario nocturno | 1.15x | Después de 7pm |

## Modelo de Datos

```sql
CREATE TABLE servicio_tiempos (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  servicio_id VARCHAR(50) NOT NULL,
  
  tiempo_base_minutos INT NOT NULL,
  valor_minuto DECIMAL(10,4) DEFAULT 1.5625,
  
  -- Calculado
  costo_tiempo DECIMAL(10,2) GENERATED ALWAYS AS 
    (tiempo_base_minutos * valor_minuto) STORED,
  
  updated_at TIMESTAMPTZ DEFAULT now()
);
```
