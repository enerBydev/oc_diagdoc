# 4.6.3 Endpoints Operador

> Endpoints exclusivos para operadores.

---

## Endpoints Agenda

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/operador/agenda` | Mi agenda |
| PATCH | `/api/operador/disponibilidad` | Cambiar disponibilidad |
| POST | `/api/operador/cita/:id/aceptar` | Aceptar cita |
| POST | `/api/operador/cita/:id/rechazar` | Rechazar cita |

---

## Endpoints Caja

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/operador/caja` | Ver caja diaria |
| POST | `/api/operador/caja/cobro` | Registrar cobro |
| GET | `/api/operador/ganancias` | Historial ganancias |

---

## Endpoints Servicio

| Método | Ruta | Descripción |
|--------|------|-------------|
| POST | `/api/operador/cita/:id/iniciar` | Iniciar servicio |
| POST | `/api/operador/cita/:id/finalizar` | Finalizar servicio |
| POST | `/api/operador/cita/:id/fotos` | Subir fotos |
| POST | `/api/operador/ubicacion` | Enviar ubicación |
