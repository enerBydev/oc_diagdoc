# 4.8.5 Feature Flags

> Toggles para habilitar/deshabilitar features.

---

## Implementación

| Aspecto | Decisión |
|---------|----------|
| Storage | Supabase `config_sistema` |
| Caching | 5 min TTL |
| UI Admin | Toggle en dashboard |

---

## Flags Activos

| Flag | Default | Descripción |
|------|---------|-------------|
| `ENABLE_CHAT` | true | Sistema de chat |
| `ENABLE_TRACKING` | true | GPS operadores |
| `ENABLE_B2B` | true | Módulo empresas |
| `ENABLE_CORPORATE` | false | Corporate+ beta |
| `ENABLE_MARKETPLACE` | false | Marketplace futuro |

---

## Uso

```typescript
const { isEnabled } = useFeatureFlags()

if (isEnabled('ENABLE_CORPORATE')) {
  // Mostrar features Corporate+
}
```
