# 7.1.3 Adaptadores Primarios (Driving Adapters)

> Conectan el mundo exterior con los casos de uso del dominio.

---

## Concepto

Los **adaptadores primarios** reciben peticiones (HTTP, eventos) y las traducen a llamadas a los puertos inbound.

---

## Tipos en OnlyCar

| Adaptador | Responsabilidad |
|-----------|-----------------|
| **API Controllers** | Endpoints REST (Nitro) |
| **Event Handlers** | Webhooks de Stripe/Gigstack |
| **Composables** | Interfaz desde Vue components |

---

## Ejemplo: Controller

```typescript
// server/api/citas/index.post.ts
export default defineEventHandler(async (event) => {
  const body = await readBody(event);
  
  // Obtener use case del contenedor DI
  const agendarCita = useContainer().get<IAgendarCita>('AgendarCita');
  
  // Ejecutar caso de uso
  const result = await agendarCita.execute({
    clienteId: event.context.user.id,
    servicioId: body.servicioId,
    ubicacion: body.ubicacion,
    fechaHora: new Date(body.fechaHora)
  });
  
  return result;
});
```
