## Cálculo de Depreciación

```typescript
const calcularDesgaste = (equipoIds: string[]): number => {
  return equipoIds.reduce((total, equipoId) => {
    const equipo = CATALOGO_EQUIPO[equipoId];
    const costoPorUso = equipo.precio / equipo.vidaUtilUsos;
    return total + costoPorUso;
  }, 0);
};
```

## Modelo de Datos

```sql
CREATE TABLE equipo_catalogo (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  nombre VARCHAR(100) NOT NULL,
  precio_compra DECIMAL(10,2) NOT NULL,
  vida_util_usos INT NOT NULL,
  costo_por_uso DECIMAL(10,4) GENERATED ALWAYS AS 
    (precio_compra / vida_util_usos) STORED,
  categoria VARCHAR(50),  -- limpieza, pulido, vapor, etc.
  activo BOOLEAN DEFAULT true
);

CREATE TABLE servicio_equipo (
  servicio_id VARCHAR(50) NOT NULL,
  equipo_id UUID REFERENCES equipo_catalogo(id),
  PRIMARY KEY (servicio_id, equipo_id)
);
```
