# 3.1.13.2.1 Cálculo de Dividendo

> Fórmula precisa para calcular dividendos trimestrales.

---

## Pseudocódigo

```typescript
interface DividendoCalculado {
  utilidadNeta: number;
  reservaLegal: number;
  fondoCrecimiento: number;
  reservas: number;
  baseDividendo: number;
  isrDividendo: number;
  dividendoNeto: number;
}

async function calcularDividendo(trimestre: number): Promise<DividendoCalculado> {
  const utilidadNeta = await getUtilidadNetaTrimestre(trimestre);
  
  // Deducciones obligatorias
  const reservaLegal = utilidadNeta * 0.05;      // 5%
  const fondoCrecimiento = utilidadNeta * 0.20;  // 20%
  const reservas = utilidadNeta * 0.20;          // 20% (contingencia + tech)
  
  // Base para dividendos
  const baseDividendo = utilidadNeta - reservaLegal - fondoCrecimiento - reservas;
  
  // Retención fiscal
  const isrDividendo = baseDividendo * 0.10;     // ISR 10%
  
  // Dividendo final
  const dividendoNeto = baseDividendo - isrDividendo;
  
  return {
    utilidadNeta,
    reservaLegal,
    fondoCrecimiento,
    reservas,
    baseDividendo,
    isrDividendo,
    dividendoNeto
  };
}
```

## Ejemplo Numérico Detallado

| Concepto | Cálculo | Valor |
|----------|---------|-------|
| Utilidad Neta Trimestral | — | $105,000 |
| - Reserva Legal (5%) | 105K × 0.05 | -$5,250 |
| - Fondo Crecimiento (20%) | 105K × 0.20 | -$21,000 |
| - Reservas (20%) | 105K × 0.20 | -$21,000 |
| **Base Dividendo (55%)** | 105K × 0.55 | **$57,750** |
| - ISR Dividendos (10%) | 57.75K × 0.10 | -$5,775 |
| **Dividendo Neto** | — | **$51,975** |

---

## Fórmula Simplificada

```
Dividendo Neto = Utilidad Neta × 0.55 × 0.90
Dividendo Neto ≈ Utilidad Neta × 0.495 (49.5%)
```
