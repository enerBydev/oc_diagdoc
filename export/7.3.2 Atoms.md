# 7.3.2 Atoms

> Los bloques de construcción más básicos del sistema de diseño.

---

## Catálogo de Atoms

| Componente | Descripción | Props Principales |
|------------|-------------|-------------------|
| `OcButton` | Botón principal | variant, size, loading |
| `OcInput` | Campo de texto | type, placeholder, error |
| `OcIcon` | Iconos (Nuxt Icon) | name, size, color |
| `OcBadge` | Etiqueta de estado | variant, size |
| `OcAvatar` | Imagen de perfil | src, size, fallback |
| `OcSpinner` | Indicador de carga | size, color |

---

## Ejemplo: OcButton

```vue
<!-- components/atoms/OcButton.vue -->
<template>
  <button 
    :class="[
      'oc-button',
      `oc-button--${variant}`,
      `oc-button--${size}`,
      { 'oc-button--loading': loading }
    ]"
    :disabled="disabled || loading"
    @click="$emit('click')"
  >
    <OcSpinner v-if="loading" size="sm" />
    <slot v-else />
  </button>
</template>

<script setup lang="ts">
defineProps<{
  variant?: 'primary' | 'secondary' | 'ghost' | 'danger';
  size?: 'sm' | 'md' | 'lg';
  loading?: boolean;
  disabled?: boolean;
}>();

withDefaults(defineProps(), {
  variant: 'primary',
  size: 'md'
});
</script>

<style scoped>
.oc-button {
  font-family: var(--oc-font-sans); /* Google Sans Flex */
  font-weight: var(--oc-font-medium);
  border-radius: var(--oc-radius-md);
  transition: all 0.2s;
}

.oc-button--primary {
  background: var(--oc-primary);
  color: white;
}

.oc-button--primary:hover {
  background: var(--oc-primary-hover);
}
</style>
```
