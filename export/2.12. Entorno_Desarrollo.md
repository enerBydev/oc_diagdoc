# 2.12 Entorno de Desarrollo (DX)

> "Optimizar la máquina del desarrollador es optimizar el producto."

---

## Visión General

Para mantener la velocidad de desarrollo mientras el equipo escala, necesitamos eliminar el "en mi máquina funciona". El entorno local debe ser determinista, rápido de levantar y capaz de funcionar sin dependencias externas costosas.

## Filosofía DX

1.  **Zero Config:** `npm run dev` debe ser suficiente para empezar.
2.  **Isolated:** No depender de la base de datos de producción ni de desarrollo compartido.
3.  **Fast Feedback:** Hot Module Replacement (HMR) siempre activo.
