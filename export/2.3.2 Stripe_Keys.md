# 2.3.2 Stripe Keys

> Credenciales para procesamiento de pagos via Stripe.

---

## Entorno de Pruebas (Test Mode)

Usar para desarrollo local y staging.

| Tipo | Clave | Descripción |
|------|-------|-------------|
| **Publishable** | `pk_test_...` | Usar en `<StripeWrapper>` frontend. |
| **Secret** | `sk_test_...` | Usar en Backend (`stripe.init`). Poder total. |
| **Webhook** | `whsec_...` | Firma para validar eventos entrantes en `/api/webhooks/stripe`. |
| **Restricted** | `rk_test_...` | (Opcional) Solo permisos de lectura. |

---

## Entorno de Producción (Live Mode)

⚠️ **ALTO RIESGO**: Acciones definitivas y movimiento de dinero real.

| Tipo | Prefijo |
|------|---------|
| **Publishable** | `pk_live_...` |
| **Secret** | `sk_live_...` |
| **Webhook** | `whsec_...` |

---

## Configuración Gigstack (Facturación)

Para la emisión automática de facturas CFDI con Gigstack.

- **App ID**: Instalar "Gigstack" desde Stripe App Marketplace.
- **API Key Gigstack**: `gs_live_...` (Configurar en dashboard Gigstack).
