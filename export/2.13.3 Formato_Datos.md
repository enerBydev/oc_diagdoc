# 2.13.3 Formato Datos

> $1,000.00 en México es diferente a 1.000,00 € en España.

---

## Intl API (Nativa)

No usamos librerías pesadas como Moment.js. Usamos el estándar web `Intl`.

### Monedas
```ts
// Composable useCurrency.ts
export const formatPrice = (amount: number, currency = 'MXN') => {
  const { locale } = useI18n()
  return new Intl.NumberFormat(locale.value, {
    style: 'currency',
    currency: currency
  }).format(amount)
}
```

### Fechas
Las fechas siempre se guardan en UTC en la base de datos y se transforman al visualizarlas.

```ts
// Formato relativo (ej. "hace 5 minutos")
const rtf = new Intl.RelativeTimeFormat(locale.value, { numeric: 'auto' });
```

---

## Zonas Horarias

Para citas y agendamiento, respetamos la zona horaria del **Servicio**, no del usuario que lo ve. Si contrato un lavado en Monterrey, la hora es Monterrey, aunque yo lo vea desde Nueva York.
