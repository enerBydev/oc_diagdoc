# 7.3.4 Organisms

> Componentes complejos que forman secciones completas de la UI.

---

## Catálogo de Organisms

| Componente | Descripción | Molecules/Atoms |
|------------|-------------|-----------------|
| `OcCitaCard` | Tarjeta de cita completa | Avatar, Badge, Button, Text |
| `OcOperadorCard` | Perfil de operador | Avatar, Rating, Badge, Button |
| `OcServicioSelector` | Selector de servicios | Cards, Checkbox, PriceTag |
| `OcPaymentMethod` | Método de pago | Icon, FormField, Button |
| `OcChatWindow` | Ventana de chat | Messages, Input, Button |
| `OcLocationPicker` | Selector de ubicación | Map, Input, Button |

---

## Ejemplo: OcCitaCard

```vue
<!-- components/organisms/OcCitaCard.vue -->
<template>
  <div class="oc-cita-card">
    <header class="oc-cita-card__header">
      <OcBadge :variant="estadoVariant">
        {{ cita.estado }}
      </OcBadge>
      <span class="oc-cita-card__date">
        {{ formatDate(cita.fechaHora) }}
      </span>
    </header>
    
    <div class="oc-cita-card__body">
      <h3 class="oc-cita-card__service">
        {{ cita.servicio.nombre }}
      </h3>
      <p class="oc-cita-card__location">
        <OcIcon name="map-pin" />
        {{ cita.ubicacion.direccion }}
      </p>
    </div>
    
    <footer class="oc-cita-card__footer">
      <OcPriceTag :amount="cita.precio" />
      <div class="oc-cita-card__actions">
        <OcButton 
          v-if="canCancel"
          variant="ghost" 
          @click="$emit('cancel')"
        >
          Cancelar
        </OcButton>
        <OcButton @click="$emit('view')">
          Ver Detalles
        </OcButton>
      </div>
    </footer>
  </div>
</template>
```
