# 2.4.3 Env Production

> ⚠️ Solo Admin/DevOps tiene acceso a estos valores.

---

## Gestión de Secretos

Las variables de producción NO existen en ningún archivo del repositorio.
Se inyectan únicamente a través del Dashboard del proveedor de hosting (Vercel/Cloudflare) o mediante `vault`.

## Variables Críticas

```bash
# Live Keys
STRIPE_SECRET_KEY=sk_live_... (Dinero real)
SUPABASE_SERVICE_KEY=... (Acceso total DB Real)

# Domains
NUXT_PUBLIC_SITE_URL=https://onlycar.mx
COOKIE_DOMAIN=.onlycar.mx
```

## Auditoría
Cualquier cambio en estas variables debe quedar registrado en el log de auditoría del proveedor de infraestructura.
