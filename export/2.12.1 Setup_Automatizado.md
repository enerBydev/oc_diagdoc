# 2.12.1 Setup Automatizado

> De `git clone` a `localhost:3000` en menos de 5 minutos.

---

## Prerrequisitos

*   Node.js 20+ (Recomendado usar `nvm`)
*   Docker & Docker Compose
*   Git

---

## Scripts de Inicio

Hemos unificado la instalación en un solo comando:

```bash
npm run init
```

Este script ejecuta internamente:
1.  `npm install`
2.  `cp .env.example .env` (Copia variables seguras por defecto)
3.  `docker-compose up -d` (Levanta base de datos local)
4.  `npm run db:migrate` (Aplica esquema SQL)
5.  `npm run db:seed` (Crea datos de prueba)

---

## Infraestructura Local (Docker)

El archivo `docker-compose.yml` levanta réplicas ligeras de los servicios cloud:

*   **Postgres:** Versión idéntica a Supabase.
*   **Redis:** Para colas locales.
*   **Mailpit:** Servidor SMTP falso para visualizar emails sin enviarlos.
