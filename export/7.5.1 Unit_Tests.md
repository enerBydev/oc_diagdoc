## Estructura

```
tests/unit/
├── domain/
│   ├── entities/
│   │   └── Cita.spec.ts
│   └── value-objects/

    └── use-cases/

```

---

## Ejemplo

```typescript
describe('Cita', () => {
  it('should create a valid cita', () => {
    const cita = new Cita('id', 'cliente', 'servicio', ubicacion, fecha);
    expect(cita.estado).toBe('pendiente');
  });
  
  it('should throw when canceling completed cita', () => {
    const cita = createCompletedCita();
    expect(() => cita.cancelar('test')).toThrow();
  });
});
```
