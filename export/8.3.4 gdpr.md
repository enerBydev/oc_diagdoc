# 8.3.4 GDPR

> Reglamento General de Protección de Datos (UE)

> [!NOTE]
> **Estado: FUTURO**
> Este documento es preparatorio. GDPR aplica cuando OnlyCar opere en Europa o procese datos de residentes EU.

---

## I. CUÁNDO APLICA GDPR

| Condición | Aplica |
|-----------|--------|
| Operaciones en EU | ✅ Sí |
| Usuarios residentes EU | ✅ Sí |
| Solo operaciones en México | ❌ No |

---

## II. DIFERENCIAS LFPDPPP VS GDPR

| Aspecto | LFPDPPP (México) | GDPR (Europa) |
|---------|------------------|---------------|
| **Ámbito territorial** | México | EU + EEE |
| **Autoridad** | Sec. Anticorrupción | DPAs nacionales |
| **Consentimiento** | Tácito permitido | **Explícito siempre** |
| **Bases legales** | Consentimiento predomina | 6 bases legales |
| **DPO** | Opcional | **Obligatorio (casos)** |
| **Portabilidad** | No explícito | **Derecho garantizado** |
| **Notificación breaches** | No regulado | **72 horas** |
| **Multas máximas** | ~$35M MXN | **€20M o 4% facturación** |
| **Edad consentimiento** | 18 años | **16 años (variable)** |

---

## III. NUEVOS DERECHOS (GDPR)

### Derechos Adicionales vs LFPDPPP

| Derecho | LFPDPPP | GDPR |
|---------|---------|------|
| Acceso | ✅ | ✅ |
| Rectificación | ✅ | ✅ |
| Cancelación/Olvido | ✅ | ✅ Ampliado |
| Oposición | ✅ | ✅ |
| **Portabilidad** | ❌ | ✅ |
| **Limitación** | ❌ | ✅ |
| **No decisiones automatizadas** | ❌ | ✅ |

---

## IV. REQUISITOS ESPECÍFICOS GDPR

### Data Protection Officer (DPO)

| Cuándo es obligatorio |
|----------------------|
| Autoridad pública |
| Tratamiento a gran escala de datos sensibles |
| Monitoreo sistemático a gran escala |

### Privacy Impact Assessment (PIA)

Obligatorio cuando el tratamiento:
- Use nuevas tecnologías
- Presente alto riesgo para derechos
- Monitoree sistemáticamente
- Trate datos a gran escala

### Notificación de Breaches

| Aspecto | Requisito |
|---------|-----------|
| Plazo | 72 horas desde detección |
| A quién | Autoridad DPA competente |
| Contenido | Naturaleza, consecuencias, medidas |
| Usuarios | Si hay alto riesgo para ellos |

---

## V. ROADMAP DE PREPARACIÓN

### Cuando decidamos expandir a Europa:

| Fase | Acción | Prioridad |
|------|--------|-----------|
| 1 | Evaluar necesidad de DPO | Alta |
| 2 | Actualizar aviso de privacidad | Alta |
| 3 | Implementar derecho de portabilidad | Alta |
| 4 | Configurar proceso de breaches | Alta |
| 5 | Realizar PIA | Media |
| 6 | Registrar representante en EU | Media |
| 7 | Revisar transferencias internacionales | Media |

---

## VI. TRANSFERENCIAS INTERNACIONALES

### Mecanismos Válidos

| Mecanismo | Aplicabilidad |
|-----------|---------------|
| Decisión de adecuación | México NO tiene |
| Cláusulas Contractuales Tipo (SCCs) | ✅ Usar para proveedores |
| Binding Corporate Rules | Solo intragrupo |
| Consentimiento explícito | Casos específicos |

### Implicación para OnlyCar

Si operamos en EU, debemos:
1. Firmar SCCs con Stripe, Gigstack, Cloudflare
2. Documentar transferencias a México
3. Evaluar riesgo adicional
