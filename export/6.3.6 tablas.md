# 6.3.6 Tablas

> Visualización de datos tabulares.

---

## Características

| Feature | Descripción |
|---------|-------------|
| **Sorting** | Click en header para ordenar |
| **Pagination** | Navegación por páginas |
| **Search** | Filtro por texto |
| **Selection** | Checkboxes para acciones masivas |
| **Responsive** | Cards en mobile |

---

## Estructura Desktop

```
┌──────────┬────────────┬─────────┬─────────┐
│ Operador │ Servicios  │ Rating  │ Acciones│
├──────────┼────────────┼─────────┼─────────┤
│ Juan P.  │ 45         │ ⭐ 4.8  │ [...]   │
│ María L. │ 32         │ ⭐ 4.9  │ [...]   │
└──────────┴────────────┴─────────┴─────────┘
        [<] Página 1 de 5 [>]
```

---

## Responsive (Mobile)

En mobile, cada fila se convierte en card:

```
┌─────────────────────┐
│ Juan Pérez          │
│ Servicios: 45       │
│ Rating: ⭐ 4.8      │
│ [Ver] [Editar]      │
└─────────────────────┘
```

## Implementación Nuxt UI

> [!IMPORTANT]
> Usar `<UTable>`. [Docs](https://ui.nuxt.com/components/table)

### Ejemplo

```vue
<UTable :data="operadores" :columns="columns">
  <template #actions-cell="{ row }">
    <UDropdownMenu :items="getActions(row)" />
  </template>
</UTable>

<script setup>
const columns = [
  { key: 'nombre', label: 'Operador', sortable: true },
  { key: 'servicios', label: 'Servicios', sortable: true },
  { key: 'rating', label: 'Rating' },
  { key: 'actions', label: '' }
]

const operadores = ref([
  { nombre: 'Juan P.', servicios: 45, rating: 4.8 },
  { nombre: 'María L.', servicios: 32, rating: 4.9 }
])
</script>
```
