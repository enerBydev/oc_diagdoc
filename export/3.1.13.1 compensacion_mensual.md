# 3.1.13.1 Compensación Mensual

> Salario mensual del fundador basado en utilidad neta y fase de crecimiento.

---

## Aplicabilidad por Fase

| Fase | Salario Real Documentado | Nota |
|------|--------------------------|------|
| **0-1** (0-10 ops) | **$0** | Reinversión total para sobrevivir |
| **2** (10-20 ops) | $0 - $10,000 | Opcional según flujo de caja |
| **2** (20-50 ops) | $10,000 - $15,000 | Transición a modelo formal |
| **3+** (50+ ops) | Fórmula completa | Aplicable 100% |

> [!NOTE]
> **En Fase 0-1, el fundador NO retira salario.**
> La supervivencia personal depende de ingresos externos o ahorros. Toda utilidad se reinvierte en escalar el proyecto.

---

## Piso Salarial Ajustado

| Escenario | Piso | Justificación |
|-----------|------|---------------|
| Ideal | $15,000 | Con 19+ operadores |
| Realista | **$10,000** | Con 28+ operadores |
| MVP | $0 | Reinversión obligatoria |

---

## Fórmula de Cálculo (Fase 3+)

```typescript
const calcularSalarioFundador = (utilidadNeta: number, fase: number): number => {
  if (fase < 2) return 0; // Fase 0-1: Sin salario
  
  const salarioBase = utilidadNeta * 0.15;
  const salarioMinimo = 10000; // Ajustado a realidad (antes 15K)
  const salarioMaximo = 80000; // Tope mensual
  
  // En transición (Fase 2), priorizar liquidez sobre salario exacto
  if (fase === 2) {
    return Math.min(salarioBase, salarioMinimo); 
  }
  
  return Math.min(Math.max(salarioBase, salarioMinimo), salarioMaximo);
};
```

## Parámetros

| Concepto | Valor | Nota |
|----------|-------|------|
| % de utilidad neta | 15% | Base de cálculo |
| Salario mínimo | **$10,000 MXN** | Piso ajustado (antes $15K) |
| Salario máximo | $80,000 MXN | Tope mensual |
| Frecuencia | Mensual | Día 15 de cada mes |
