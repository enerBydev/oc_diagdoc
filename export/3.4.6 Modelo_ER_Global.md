## Diagrama Maestro

```mermaid
erDiagram
    USERS ||--o{ PERFILES_OPERADOR : tiene
    USERS ||--o{ PERFILES_CLIENTE : tiene
    
    PERFILES_OPERADOR ||--o{ SERVICIOS_ASIGNADOS : ejecuta
    PERFILES_CLIENTE ||--o{ SOLICITUDES_SERVICIO : crea
    
    SOLICITUDES_SERVICIO ||--|| PAGOS : genera
    
    TABLA_PRECIOS ||--o{ SOLICITUDES_SERVICIO : cotiza
    
    PERFILES_OPERADOR {
        uuid id PK
        string estado_operativo
        jsonb ubicacion_actual
        decimal saldo_caja
    }

    SOLICITUDES_SERVICIO {
        uuid id PK
        uuid cliente_id FK
        uuid operador_id FK
        string estado
        timestamp fecha_programada
        decimal precio_final
    }
    
    PAGOS {
        uuid id PK
        uuid solicitud_id FK
        string metodo
        string estado
        string proveedor_ref
    }
    
    INVENTARIO_OPERADOR }|--|| PERFILES_OPERADOR : posee
```
