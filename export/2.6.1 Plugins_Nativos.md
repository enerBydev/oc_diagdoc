## Matriz de Plugins Core

| Plugin | Package | Uso Principal | Fallback Web |
|--------|---------|---------------|--------------|
| **Camera** | `@capacitor/camera` | Fotos de evidencia (Operador) | `<input type="file">` |
| **Geolocation** | `@capacitor/geolocation` | Ubicación tiempo real, Tracking | Geolocation API |
| **Push** | `@capacitor/push-notifications` | Avisos de servicio, Chat | Service Worker |
| **Storage** | `@capacitor/preferences` | Persistencia Auth, Settings | LocalStorage |
| **Device** | `@capacitor/device` | ID único, Info batería | `navigator` |
| **App** | `@capacitor/app` | Estado background/foreground | Page Visibility API |
| **Status Bar** | `@capacitor/status-bar` | Estilo Dark Mode nativo | Meta tags |
| **Splash Screen**| `@capacitor/splash-screen` | Carga inicial branding | CSS Loader |

## Configuración Especial

### Configuración en `capacitor.config.ts`

```ts
import { CapacitorConfig } from '@capacitor/cli';

const config: CapacitorConfig = {
  appId: 'com.onlycar.app',
  appName: 'OnlyCar',
  webDir: 'dist',
  plugins: {
    PushNotifications: {
      presentationOptions: ["badge", "sound", "alert"],
    },
    SplashScreen: {
      launchShowDuration: 2000,
      backgroundColor: "#293133"
    }
  }
};

export default config;
```

---

## Política de Plugins

1. **Prioridad Oficial:** Usar siempre plugins `@capacitor/*` oficiales.
2. **Plugins Comunidad:** Solo si no existe oficial, usar [Capawesome](https://github.com/capawesome-team) o verificados.
3. **Auditoría:** Revisar updates mensualmente para evitar breaking changes con versiones nuevas de iOS/Android.
