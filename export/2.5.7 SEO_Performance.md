# 2.5.7 SEO y Performance

> Maximizar la visibilidad en motores de búsqueda y la velocidad percibida por el usuario.

---

## Estrategia SEO (Search Engine Optimization)

Dado que OnlyCar tiene componentes públicos (Landing, Cotizador Anónimo, Blog), el SEO es crítico.

### Metatags Dinámicos
Usamos `useSeoMeta` en cada página para definir títulos, descripciones e imágenes OG.

```typescript
useSeoMeta({
  title: 'Lavado de Autos a Domicilio - OnlyCar',
  ogTitle: 'OnlyCar - Tu auto limpio en minutos',
  description: 'Solicita servicio de lavado profesional a domicilio en Nuevo Laredo.',
  ogDescription: 'Solicita servicio de lavado profesional a domicilio en Nuevo Laredo.',
  ogImage: 'https://onlycar.mx/og-cover.jpg',
  twitterCard: 'summary_large_image',
})
```

### Sitemap y Robots
- **@nuxtjs/sitemap**: Generación automática de `sitemap.xml` basado en rutas estáticas y dinámicas (API).
- **@nuxtjs/robots**: Control de `robots.txt`.
    - `Allow: /` (Público)
    - `Disallow: /admin`
    - `Disallow: /app` (Dashboard privado)

### Schema.org (JSON-LD)
Implementar datos estructurados para "LocalBusiness" y "Service" mediante `useSchemaOrg` (nuxt-schema-org) para Rich Snippets en Google.

---

## Performance (Core Web Vitals)

### LCP (Largest Contentful Paint)
- Pre-cargar imagen Hero (`<link rel="preload">`).
- Optimizar imágenes con **Nuxt Image**: Formato WebP/AVIF automático y tamaños responsivos.

### CLS (Cumulative Layout Shift)
- Definir dimensiones explícitas `width` y `height` en imágenes.
- Reservar espacio para anuncios o banners dinámicos.
- Usar `font-display: swap` en fuentes web.

### INP (Interaction to Next Paint)
- Evitar tareas largas en el Main Thread.
- Deferir Javascript no crítico.

---

## Carga Diferida (Lazy Loading)

### Componentes
Nuxt auto-importa componentes de manera perezosa si se prefijan con `Lazy`.
```vue
<LazyMapWidget v-if="showMap" />
```
*El código del mapa solo se descarga cuando `showMap` es true.*

### Rutas
El router de Nuxt hace code-splitting automático por página. Cada ruta `pages/` genera un chunk JS separado.

---

## Optimización de Fuentes

Usar **@nuxt/fonts** para descargar, optimizar y servir fuentes de Google Fonts localmente, eliminando layout shifts y peticiones a terceros.
- Familia: `Inter` (UI) y `Outfit` (Headings).
