# 3.2.6.3 RLS Pagos

> Policies para pagos, facturas_emitidas, comisiones.

---

## Políticas

```sql
-- Clientes ven sus propios pagos
CREATE POLICY "pagos_cliente_own"
  ON pagos FOR SELECT
  USING (cliente_id = auth.uid());

-- Operadores ven comisiones propias
CREATE POLICY "comisiones_operador_own"
  ON comisiones_operador FOR SELECT
  USING (operador_id = auth.uid());
```
