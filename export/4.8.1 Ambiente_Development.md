# 4.8.1 Ambiente Development

> Configuración para desarrollo local.

---

## Requisitos

| Componente | Versión |
|------------|---------|
| Node.js | 20.x LTS |
| pnpm | 8.x |
| Docker | (opcional) |

---

## Configuración

```bash
# Clonar repo
git clone git@github.com:enerBydev/onlycar_plataform.git
cd ./Github_Proyectos/OnlyServicesMX/OnlyCar/repo_OnlyCar/OnlyCar_Plataform

# Instalar deps
pnpm install

# Variables de entorno
cp .env.example .env.local

# Iniciar dev server
pnpm dev
```
