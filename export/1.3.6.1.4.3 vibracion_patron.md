# 1.3.6.1.4.3 Patrón de Vibración

Sistema de vibración háptica para solicitudes de servicio.

---

## Patrones de Vibración

### Solicitud Estándar ("Da-Da-Daaaa")

```
Array: [0, 500, 200, 500, 200, 500, 500, 1000]
Repetición: 3 veces (~10 segundos)
```

### Solicitud Urgente (Pulsos rápidos)

```
Array: [0, 200, 100, 200, 100, 200, 100, 200, 100, 200]
Repetición: 4 veces (~12 segundos)
```

### Cliente Favorito (Doble + largo)

```
Array: [0, 150, 50, 150, 150, 1000]
Repetición: 2 veces (~8 segundos)
```

---

## Implementación Android

```kotlin
object VibrationPatterns {
    val STANDARD = longArrayOf(0, 500, 200, 500, 200, 500, 500, 1000)
    val URGENT = longArrayOf(0, 200, 100, 200, 100, 200, 100, 200, 100, 200)
    val FAVORITE = longArrayOf(0, 150, 50, 150, 150, 1000)
}

fun vibrateForRequest(type: RequestType) {
    val pattern = when (type) {
        RequestType.STANDARD -> VibrationPatterns.STANDARD
        RequestType.URGENT -> VibrationPatterns.URGENT
        RequestType.FAVORITE -> VibrationPatterns.FAVORITE
    }
    vibrator.vibrate(VibrationEffect.createWaveform(pattern, -1))
}
```
