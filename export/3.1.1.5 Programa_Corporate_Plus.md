# 3.1.1.5 Programa Corporate+

> Beneficios exclusivos para empleados de empresas cliente B2B.

---

## Beneficios del Programa

| Beneficio | Valor |
|-----------|-------|
| **Descuento permanente** | 15% (código `CORP15`) |
| **Prioridad en agenda** | 48 horas antes que público |
| **Tarjetas de referido** | 10 tarjetas físicas |
| **Facturación** | A nombre de la empresa |

---

## Elegibilidad

### Requisitos
1. Ser empleado de una empresa cliente B2B activa
2. Verificar email corporativo (@empresa.com)
3. Estar en lista autorizada de la empresa

### Verificación
```typescript
const verificarCorporatePlus = async (email: string) => {
  // 1. Extraer dominio del email
  const dominio = email.split('@')[1];
  
  // 2. Buscar empresa por dominio
  const empresa = await supabase
    .from('empresas')
    .select('*')
    .eq('dominio_email', dominio)
    .eq('corporate_plus_activo', true)
    .single();
  
  return empresa ? true : false;
};
```

---

## Prioridad de Agenda

| Tipo Usuario | Acceso a Agenda |
|--------------|-----------------|
| Corporate+ | 48 horas antes |
| Público General (B2C) | 72 horas antes |
| B2B Empresa | Según contrato |

---

## Código de Referido

**Formato:** `NLD-B-{alfanumerico:6}`  
**Ejemplo:** `NLD-B-F7M2P4`

Cada empleado Corporate+ puede referir a otros empleados o clientes externos usando su código personal.
