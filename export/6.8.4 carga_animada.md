# 6.8.4 Carga Animada

| Tipo | Animación |
|------|-----------|
| Spinner | rotate infinite |
| Progress | width transition |
| Skeleton | shimmer gradient |

## Implementación @vueuse/motion

> [!IMPORTANT]
> Combinar con Nuxt UI `<USkeleton>` para estados de carga.

### Spinner con Motion

```vue
<div
  v-motion
  :animate="{ rotate: 360 }"
  :transition="{ duration: 1000, repeat: Infinity, ease: 'linear' }"
>
  <Icon name="i-heroicons-arrow-path" size="24" />
</div>
```

### Progress Animado

```vue
<div class="bg-neutral-700 rounded-full h-2 overflow-hidden">
  <div
    v-motion
    :initial="{ width: '0%' }"
    :animate="{ width: `${progress}%` }"
    :transition="{ duration: 300 }"
    class="bg-primary h-full"
  />
</div>
```

### Skeleton Shimmer (CSS)

```css
.skeleton-shimmer {
  background: linear-gradient(
    90deg,
    #293133 25%,
    #3a4042 50%,
    #293133 75%
  );
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}

@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```
