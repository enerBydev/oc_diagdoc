## Tipos de DTD

| Tipo | Herramienta | Output |
|------|-------------|--------|
| API Spec | OpenAPI 3.0 | `docs/api/openapi.yaml` |
| Components | Storybook | Static site |
| Types | TypeDoc | HTML docs |
| ADRs | Markdown | `docs/adr/*.md` |

---

## OpenAPI

```yaml
# docs/api/openapi.yaml
openapi: 3.0.3
info:
  title: OnlyCar API
  version: 1.0.0
paths:
  /api/citas:
    post:
      summary: Crear nueva cita
      requestBody:
        content:
          application/json:
            schema:
              $ref: '#/components/schemas/CreateCitaRequest'
      responses:
        '201':
          description: Cita creada
```
