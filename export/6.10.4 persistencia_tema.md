# 6.10.4 Persistencia Tema

## Prioridad

1. `localStorage.theme`
2. `prefers-color-scheme`
3. Default: dark

## Implementación @nuxtjs/color-mode

> [!IMPORTANT]
> Usar `useColorMode()`. [Docs](https://color-mode.nuxtjs.org/)
> Incluido automáticamente con Nuxt UI.

### Uso

```vue
<script setup>
const colorMode = useColorMode()
</script>

<template>
  <UButton
    :icon="colorMode.value === 'dark' ? 'i-heroicons-sun' : 'i-heroicons-moon'"
    @click="colorMode.preference = colorMode.value === 'dark' ? 'light' : 'dark'"
  />
</template>
```

### Config nuxt.config.ts

```ts
export default defineNuxtConfig({
  colorMode: {
    preference: 'dark',      // Default
    fallback: 'dark',
    classSuffix: '',
    storageKey: 'oc-color-mode'
  }
})
```

### CSS Variables

```css
:root {
  --color-background: theme('colors.white');
  --color-text: theme('colors.neutral.900');
}

.dark {
  --color-background: theme('colors.neutral.900');
  --color-text: theme('colors.white');
}
```
