# 1.4.3.5 Benchmarking

## Descripción General

Este módulo documenta el proceso de benchmarking de OnlyCar contra la industria de detailing automotriz y servicios on-demand similares. El objetivo es contextualizar las métricas de [[Proyecto OnlyCarNLD/Datos/1.4.3 analisis_mejora]] contra estándares externos para identificar oportunidades de mejora.

## Fuentes de Datos

### Industria de Detailing Automotriz

| Fuente | Tipo | Frecuencia |
|--------|------|------------|
| **IDA (International Detailing Association)** | Encuestas anuales | Anual |
| **SEMA Show Reports** | Tendencias de mercado | Anual |
| **Encuestas propias a competidores** | Mystery shopping | Trimestral |
| **Google Reviews agregados** | Análisis de sentimiento | Mensual |

### Plataformas On-Demand Comparables

| Plataforma | Métricas Públicas | Relevancia |
|------------|-------------------|------------|
| **Uber** | NPS, driver ratings | Alta (modelo similar) |
| **DiDi** | Calificaciones, tiempos | Alta (modelo similar) |
| **Rappi/DoorDash** | Entrega, satisfacción | Media |
| **TaskRabbit** | Calificación de tareas | Media |

## Métricas Benchmark

### NPS (Net Promoter Score)

| Categoría | Benchmark Industria | Meta OnlyCar |
|-----------|---------------------|-----------------|
| Detailing tradicional | 35-45 | - |
| Detailing móvil | 40-50 | - |
| Apps on-demand | 45-55 | - |
| **OnlyCar Target** | - | **>50** |

### CSAT (Customer Satisfaction)

| Categoría | Benchmark | Meta OnlyCar |
|-----------|-----------|-----------------|
| Servicios automotrices | 4.0/5.0 | - |
| Apps de servicios | 4.3/5.0 | - |
| **OnlyCar Target** | - | **>4.5/5.0** |

### Tasa de Respuesta a Encuestas

| Categoría | Benchmark | Meta OnlyCar |
|-----------|-----------|-----------------|
| Email post-servicio | 10-15% | - |
| Push notification | 20-30% | - |
| In-app prompt | 35-45% | - |
| **OnlyCar Target** | - | **>40%** |

## Metas por Temporada

### Temporada Alta (Marzo-Mayo, Octubre-Diciembre)

| Métrica | Meta Normal | Meta Temporada Alta |
|---------|-------------|---------------------|
| NPS | 50 | 55 |
| CSAT | 4.5 | 4.6 |
| Tiempo respuesta | 2h | 1.5h |
| Tasa cancelación operador | <5% | <3% |

### Temporada Baja (Junio-Septiembre, Enero-Febrero)

| Métrica | Meta Normal | Meta Temporada Baja |
|---------|-------------|---------------------|
| NPS | 50 | 48 (tolerable) |
| CSAT | 4.5 | 4.4 (tolerable) |
| Tiempo respuesta | 2h | 3h (aceptable) |
| Promociones activas | Base | +20% descuentos |

## Análisis de Tendencias

### Indicadores de Alerta Temprana

| Señal | Umbral | Acción |
|-------|--------|--------|
| NPS cae 5+ puntos en 2 semanas | <45 | Investigación inmediata |
| CSAT cae bajo 4.2 | <4.2 | Revisión de operadores |
| Tasa respuesta cae 10% | <30% | Revisar timing de encuestas |
| Reviews negativos suben 20% | +20% | Análisis de causas raíz |

## Proceso de Benchmarking

### Frecuencia de Análisis

| Tipo | Frecuencia | Responsable |
|------|------------|-------------|
| Snapshot rápido | Semanal | Sistema automático |
| Análisis completo | Mensual | Admin |
| Comparativa industria | Trimestral | Admin + Consultor |
| Reporte ejecutivo B2B | Bajo demanda | [[Proyecto OnlyCarNLD/Datos/1.4.5 reportes_ejecutivos]] |

### Flujo de Información

1. Sistema recopila métricas internas diariamente
2. Agregación semanal automática
3. Comparación mensual contra benchmarks
4. Alertas si métricas salen de rango
5. Reporte trimestral con recomendaciones

## Limitaciones

- Datos de competidores a menudo son estimados (no públicos)
- Industria de detailing fragmentada, pocos datos estandarizados
- Comparación con apps on-demand es aproximada (servicios diferentes)
