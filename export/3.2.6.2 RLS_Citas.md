# 3.2.6.2 RLS Citas

> Policies para citas, fotos_servicio.

---

## Políticas

```sql
-- Clientes ven sus propias citas
CREATE POLICY "citas_cliente_own"
  ON citas FOR SELECT
  USING (cliente_id = auth.uid());

-- Operadores ven citas asignadas
CREATE POLICY "citas_operador_assigned"
  ON citas FOR SELECT
  USING (operador_id = auth.uid());

-- Admin ve todas
CREATE POLICY "citas_admin_all"
  ON citas FOR ALL
  USING (auth.user_role() = 'admin');
```
