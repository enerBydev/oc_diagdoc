# 6.8.2 Feedback Animado

| Acción | Animación |
|--------|-----------|
| Button press | scale(0.98) |
| Success | checkmark + green |
| Error | shake |

## Implementación @vueuse/motion

> [!IMPORTANT]
> Usar `v-motion:tap` para feedback de interacción.

### Button Press

```vue
<UButton
  v-motion
  :tap="{ scale: 0.98 }"
  label="Presionar"
/>
```

### Success Animation

```vue
<div
  v-motion
  :initial="{ scale: 0, opacity: 0 }"
  :enter="{ scale: 1, opacity: 1 }"
  class="text-green-500"
>
  <Icon name="i-heroicons-check-circle" size="48" />
</div>
```

### Shake (Error)

```vue
<script setup>
import { useMotion } from '@vueuse/motion'

const shakeRef = ref(null)
const { apply } = useMotion(shakeRef, {
  initial: { x: 0 },
  shake: { x: [-10, 10, -10, 10, 0] }
})

function triggerShake() {
  apply('shake')
}
</script>

<template>
  <div ref="shakeRef">
    <UInput v-model="value" :class="{ 'border-red-500': hasError }" />
  </div>
</template>
```
