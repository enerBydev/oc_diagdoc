# 6.7.2 Estados Vacíos

| Contexto | Mensaje | CTA |
|----------|---------|-----|
| Sin citas | "No tienes citas" | Agendar |
| Sin vehículos | "Agrega tu auto" | + Vehículo |
| Sin resultados | "Sin resultados" | Limpiar filtros |

## Implementación Custom: OcEmptyState

> [!WARNING]
> Requiere componente custom `OcEmptyState.vue` (no existe en Nuxt UI)

### Componente

```vue
<!-- components/OcEmptyState.vue -->
<script setup lang="ts">
defineProps<{
  icon?: string
  title: string
  description?: string
  actionLabel?: string
}>()

const emit = defineEmits(['action'])
</script>

<template>
  <div class="flex flex-col items-center justify-center py-12 text-center">
    <Icon v-if="icon" :name="icon" size="64" class="text-neutral-500 mb-4" />
    <h3 class="text-lg font-semibold mb-2">{{ title }}</h3>
    <p v-if="description" class="text-neutral-400 mb-6">{{ description }}</p>
    <UButton v-if="actionLabel" @click="emit('action')">{{ actionLabel }}</UButton>
  </div>
</template>
```

### Uso

```vue
<OcEmptyState
  icon="i-heroicons-calendar"
  title="No tienes citas"
  description="Agenda tu primer servicio"
  action-label="Agendar ahora"
  @action="navigateTo('/app/citas/nueva')"
/>
```
