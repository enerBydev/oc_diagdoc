# 2.5 Arquitectura Frontend (Nuxt)

> Definición técnica de la arquitectura Nuxt para OnlyCar.

---

## Visión General

El frontend de OnlyCar está construido sobre **Nuxt 4** utilizando el paradigma de **Modular Architecture**. El objetivo es mantener el código desacoplado, testearble y escalable para una aplicación que funciona tanto en Web como en Mobile (Capacitor).
