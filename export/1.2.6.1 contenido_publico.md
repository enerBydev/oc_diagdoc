## Rutas Frontend

```typescript
// middleware/auth.ts
export const publicRoutes = [
  '/',
  '/servicios',
  '/servicios/:slug',
  '/precios',
  '/cobertura',
  '/testimonios',
  '/faq',
  '/cotizar',
  '/contacto',
  '/blog',
  '/blog/:slug',
  '/privacidad',
  '/terminos',
  '/login',
  '/registro',
];

export const protectedRoutes = [
  '/dashboard',
  '/agendar',
  '/historial',
  '/perfil',
  '/vehiculos',
  '/chat',
];

// Middleware de autenticación
export default defineNuxtRouteMiddleware((to) => {
  const user = useSupabaseUser();
  
  if (protectedRoutes.some(r => to.path.startsWith(r)) && !user.value) {
    return navigateTo('/login?redirect=' + to.path);
  }
});
```

## Datos Expuestos Públicamente

### ✅ Exponer

| Dato | Justificación |
|------|---------------|
| Lista de servicios | SEO, catálogo |
| Precios base | Transparencia |
| Zonas de cobertura | Decisión informada |
| Testimonios aprobados | Confianza |
| FAQ | Soporte self-service |

### ❌ NO Exponer

| Dato | Razón |
|------|-------|
| Descuentos corporativos | Exclusivo B2B |
| Disponibilidad en tiempo real | Incentiva registro |
| Datos de operadores | Privacidad |
| Métricas internas | Confidencial |
| Precios personalizados | Requiere perfil |
