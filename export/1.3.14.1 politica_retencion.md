# 1.3.14.1 Política de Retención

> Cuánto tiempo guardamos cada tipo de dato.

---

## Matriz de Retención

| Categoría | Tipo de Dato | Activo | Archivo | Total | Justificación |
|-----------|--------------|--------|---------|-------|---------------|
| **Mensajes** | Texto plano | 1 año | 5 años | 6 años | LFPDPPP Art. 37 |
| **Mensajes** | CARDs interactivos | 1 año | 5 años | 6 años | Auditoría |
| **Multimedia** | Fotos de servicio | 6 meses | 2 años | 2.5 años | Evidencia |
| **Multimedia** | Notas de voz | 6 meses | 2 años | 2.5 años | Evidencia |
| **Multimedia** | Documentos PDF | 1 año | 5 años | 6 años | Contratos |
| **Metadatos** | Timestamps, IDs | Indefinido | - | Indefinido | Integridad |
| **Logs** | Acceso a datos | 1 año | 3 años | 4 años | Auditoría |

---

## Justificación Legal

### LFPDPPP (México)

> **Artículo 37:** Los datos personales deben ser mantenidos de manera que
> permitan el ejercicio del derecho de acceso, y serán cancelados cuando
> hayan dejado de ser necesarios para la finalidad para la cual fueron recabados.

**Interpretación aplicada:**
- Mensajes son necesarios para soporte y disputas → 6 años total
- Multimedia es evidencia del servicio → 2.5 años suficiente
- Metadatos no son datos personales → Indefinido

### Aviso de Privacidad

El aviso de privacidad de OnlyCar debe incluir:

```markdown
## Retención de Datos

Conservamos tus datos de la siguiente manera:
- Mensajes de chat: Hasta 6 años desde el servicio
- Fotos y multimedia: Hasta 2.5 años desde el servicio
- Datos de cuenta: Mientras tu cuenta esté activa

Puedes solicitar la eliminación de tus datos en cualquier momento
enviando un correo a privacidad@onlycar.mx
```

## Configuración Técnica

```typescript
// config/retention-policy.ts
export const RETENTION_POLICY = {
  messages: {
    text: { active: 365, archive: 1825, total: 2190 }, // días
    cards: { active: 365, archive: 1825, total: 2190 },
  },
  multimedia: {
    photos: { active: 180, archive: 730, total: 910 },
    audio: { active: 180, archive: 730, total: 910 },
    documents: { active: 365, archive: 1825, total: 2190 },
  },
  metadata: {
    timestamps: { active: Infinity, archive: 0, total: Infinity },
    ids: { active: Infinity, archive: 0, total: Infinity },
  },
  logs: {
    access: { active: 365, archive: 1095, total: 1460 },
  },
};

// Helper
export const getDeletionDate = (createdAt: Date, type: string, subtype: string): Date => {
  const days = RETENTION_POLICY[type][subtype].total;
  return new Date(createdAt.getTime() + days * 24 * 60 * 60 * 1000);
};
```

## Excepciones

| Caso | Retención Extendida | Razón |
|------|---------------------|-------|
| Disputa activa | Hasta resolución + 1 año | Evidencia |
| Solicitud legal | Según requerimiento | Obligación |
| Solicitud de usuario | Eliminación inmediata | Derecho ARCO |
