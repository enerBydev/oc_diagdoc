# 3.5.1 Escenarios Monte Carlo

> **Objetivo:** Convertir la incertidumbre en rangos de confianza accionables.

---

## Variables Aleatorias de Entrada

| Variable            | Distribución | Parámetros Ejemplo        | Origen    |
| ------------------- | ------------ | ------------------------- | --------- |
| **Demanda Diaria**  | Poisson      | $\lambda = 15$ servicios  | Histórico |
| **Clima (Lluvia)**  | Bernoulli    | $p = 0.15$ (prob lluvia)  | API Clima |
| **Eficiencia Op**   | Normal       | $\mu = 1.0, \sigma = 0.1$ | Perfil Op |
| **No-Show Cliente** | Binomial     | $n=citas, p=0.05$         | Histórico |

---

## Ejecución de Simulación

1.  **Definir Horizonte:** 30 días (Mes Operativo).
2.  **Iteraciones:** 10,000 runs.
3.  **Cálculo por Run:**
    *   Generar clima (Lluvia/Sol).
    *   Generar demanda base afectada por clima.
    *   Simular asignación y ejecución (costos + ingresos).
    *   Calcular Net Profit.

---

## Visualización de Salida (Histogramas)

El sistema genera un objeto JSON con los percentiles clave:
*   **P10 (Pesimista):** Ingreso mínimo con 90% certeza.
*   **P50 (Base):** Escenario más probable.
*   **P90 (Optimista):** Potencial máximo realista.
