# 2.17 Estrategia de Performance

> "La velocidad es una feature". Estrategia centralizada para garantizar tiempos de carga y respuesta óptimos.

---

## Principios de Performance

1.  **Mobile First:** Optimización prioritaria para redes 4G inestables y dispositivos gama media.
2.  **Offline First:** La aplicación debe cargar instantáneamente desde Service Worker (Workbox).
3.  **Core Web Vitals:** No son opcionales. El incumplimiento rompe el pipeline de CI/CD.
4.  **Zero Shift:** El layout no debe moverse mientras carga ([[Proyecto OnlyCarNLD/Datos/2.17.1 Core_Vitals_Budget]]).
