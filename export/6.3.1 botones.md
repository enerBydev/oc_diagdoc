# 6.3.1 Botones

> Componente base para acciones del usuario.

---

## Variantes

| Variante | Uso | Fondo | Texto |
|----------|-----|-------|-------|
| **primary** | CTA principal | `#E87C1F` | `#FFFFFF` |
| **secondary** | Acción secundaria | `#5E6365` | `#FFFFFF` |
| **outline** | Acción terciaria | Transparente | `#E87C1F` |
| **ghost** | Acción sutil | Transparente | `#5E6365` |
| **danger** | Eliminar/Cancelar | `#EF4444` | `#FFFFFF` |

---

## Estados

| Estado | Descripción | Efecto |
|--------|-------------|--------|
| **Default** | Estado inicial | — |
| **Hover** | Mouse sobre | `brightness(1.1)` |
| **Active** | Click | `scale(0.98)` |
| **Disabled** | Deshabilitado | `opacity: 0.5` |
| **Loading** | Procesando | Spinner + texto |

---

## Tamaños

| Token | Height | Padding | Font Size |
|-------|--------|---------|-----------|
| `sm` | 32px | 12px 16px | 14px |
| `md` | 40px | 12px 24px | 16px |
| `lg` | 48px | 16px 32px | 18px |

## Implementación Nuxt UI

> [!IMPORTANT]
> Usar `<UButton>` de Nuxt UI. [Docs](https://ui.nuxt.com/components/button)

### Ejemplos

```vue
<!-- Primary -->
<UButton color="primary" label="Solicitar Servicio" />

<!-- Secondary -->
<UButton color="neutral" variant="soft" label="Cancelar" />

<!-- Outline -->
<UButton color="primary" variant="outline" label="Ver más" />

<!-- Ghost -->
<UButton color="neutral" variant="ghost" label="Omitir" />

<!-- Danger -->
<UButton color="error" label="Eliminar" />

<!-- Loading -->
<UButton color="primary" loading label="Procesando..." />

<!-- Sizes -->
<UButton size="sm" label="Pequeño" />
<UButton size="md" label="Normal" />
<UButton size="lg" label="Grande" />
```

### Config `app.config.ts`

```ts
export default defineAppConfig({
  ui: {
    button: {
      defaultVariants: {
        color: 'primary'
      }
    }
  }
})
```
