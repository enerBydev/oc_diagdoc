# 2.16.1 Internal Event Bus

> Un canal centralizado y tipado para toda la aplicación.

---

## Librería: Mitt

Usamos `mitt` (200 bytes) por su simplicidad y soporte TS.

### Definición de Eventos (Types)

```ts
// utils/bus.ts
type Events = {
  'user:login': { userId: string; role: string };
  'service:created': { serviceId: string };
  'payment:success': { amount: number; transactionId: string };
};

export const bus = mitt<Events>();
```

### Uso

```ts
// Componente A
bus.emit('service:created', { serviceId: '123' });

// Componente B (Global Listener)
bus.on('service:created', (payload) => {
  refreshDashboard(payload.serviceId);
});
```
